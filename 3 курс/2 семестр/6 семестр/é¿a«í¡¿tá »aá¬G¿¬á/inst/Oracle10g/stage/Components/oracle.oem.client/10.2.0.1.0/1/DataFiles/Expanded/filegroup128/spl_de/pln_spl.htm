Plan-Editor
 
Der Plan-Editor ist eine Komponente von Change Manager, die gestartet wird, wenn Sie einen Änderungsplan erstellen oder bearbeiten. Danach definieren Sie jedes Element in dem Änderungsplan von oben nach unten in der Baumstruktur von Plan-Editor:
  1. Erstellen Sie Änderungsanforderungen.
  2. Geben Sie Zieldatenbanken an (in denen Sie die Änderungen vornehmen).
  3. Generieren Sie Ihr Skript.
  4. Führen Sie das Skript aus.
