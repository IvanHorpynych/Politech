Gestión de Replicación
 
La replicación es el proceso de copiar y mantener objetos de bases de datos, como tablas, en varias bases de datos que conforman un sistema de base de datos distribuido.
Utilice la Consola para realizar:
Replicación de varios maestros seleccionando Replicación de Varios Maestros en el navegador.
Replicación de vista materializada seleccionando Replicación de Vista Materializada en el navegador.
