Warehouse Management
 
Ein Data Warehouse ist eine relationale Datenbank, die eher zur Abfrage und Analyse als zur Transaktionsverarbeitung ausgelegt ist. Oracle unterstützt die komplexe Analyse von Warehouse-Daten durch Anwendungen zur analytischen Online-Verarbeitung (OLAP). Oracle stellt außerdem ein Verfahren bereit, mit dem die Performance Ihres Warehouses mit Hilfe von Zusammenfassungen verbessert wird.
 
Wählen Sie den Ordner Verwaltung der Zusammenfassung im Navigator zur Verwaltung von Zusammenfassungen.
