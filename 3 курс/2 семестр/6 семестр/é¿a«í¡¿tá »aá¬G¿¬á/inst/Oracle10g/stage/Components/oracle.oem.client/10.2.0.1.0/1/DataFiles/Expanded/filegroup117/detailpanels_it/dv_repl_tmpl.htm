Modelli di distribuzione
Replica di viste materializzate>Modelli di distribuzione
I modelli di distribuzione consentono di definire una collection di definizioni di viste materializzate presso un sito principale e utilizzare parametri nelle definizioni in modo che le viste materializzate possano essere personalizzate per singoli utenti.
Per creare un modello di distribuzione presso un sito principale, effettuare le operazioni riportate di seguito.
  1. Fare clic su Imposta siti principali... per impostare il sito principale, se non lo si è ancora fatto.
  2. Se si desidera includere viste materializzate aggiornabili nel modello, le relative tabelle di base devono trovarsi nel gruppo principale.
    Fare clic su Crea gruppo principale... per creare il gruppo principale che gestirà gli oggetti replicati.
  3. Fare clic su Crea modello... per avviare la procedura guidata di creazione del modello di distribuzione.
Per personalizzare il modello per singoli utenti e distribuirlo sui siti di viste materializzate, effettuare le operazioni riportate di seguito.
  1. Fare clic su Generazione script di modello... per completare la generazione degli script di modello fuori linea.
  2. Distribuire i file di creazione delle istanze a ciascun utente interessato. Gli utenti possono eseguire lo script sul sito di viste materializzate utilizzando SQL*Plus per creare l'ambiente di viste materializzate.
Per ulteriori informazioni sui modelli di distribuzione, fare clic sul pulsante ?.
