Server delle directory
 
La cartella Server delle directory elenca tutti i server delle directory rilevati da Console, visualizzandone le proprietà. Fare clic con il pulsante destro del mouse su un server delle directory per effettuare le operazioni riportate di seguito.
Avviare o chiudere il server delle directory.
Effettuare il ping del server per verificare che sia in funzione.
Visualizzare le proprietà del server, ad esempio la versione.
Eliminare il server delle directory dalla struttura del Navigator.
Pianificare periodi di sospensione delle notifiche oppure sospendere tutte le notifiche per un server specifico.
È inoltre possibile creare eventi per monitorare i server delle directory e individuare le condizioni che richiedono un intervento, nonché pianificare job per correggere tali condizioni.
