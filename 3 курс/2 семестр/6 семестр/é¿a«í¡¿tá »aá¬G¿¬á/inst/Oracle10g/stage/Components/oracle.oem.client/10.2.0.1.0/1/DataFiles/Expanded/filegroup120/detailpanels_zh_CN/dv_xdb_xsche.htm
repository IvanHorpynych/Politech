基于 XML 方案的对象
XML 数据库管理 > 基于 XML 方案的对象
XML 方案是一种方案定义语言 (也可以在 XML 中找到), 它可用于描述一致的 XML 例程文档的结构和语义。
使用 Oracle 9.2, 您可以执行以下操作:
通过使用现有的 XML 数据库资源，或使用可以本地访问或通过 Internet 访问的 XML 数据库方案文件，在数据库中注册 XML 方案。
根据 XML 方案创建结构化存储。
您可以按如下两种方法之一执行：
创建一个表, 该表基于 XML 方案中定义的结构
创建一个表, 该表中的列基于 XML 方案中定义的结构
创建基于 XML 方案的存储的优点包括:
  • 此存储中存储的 XML 例程文档占用极少的空间
  • 高度优化了对通过 XPath 表示的数据块的访问
  • 极大地改善了对 XML 例程文档的一部分所进行的更新操作
通过创建基于 XML 方案的视图, 使现有数据遵循 XML 格式
还可以通过使用 XPath 表达式, 为基于 XML 方案的表创建函数索引。
