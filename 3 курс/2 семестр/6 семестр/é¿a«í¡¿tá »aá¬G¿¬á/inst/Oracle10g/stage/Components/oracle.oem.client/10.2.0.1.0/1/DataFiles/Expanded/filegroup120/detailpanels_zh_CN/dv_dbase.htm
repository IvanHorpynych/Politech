数据库
 
数据库文件夹包含您可以访问的所有数据库。您可以展开指定的数据库, 以管理下列对象:
例程，包括初始化例程。
方案, 包括表, 索引和所有其他方案对象。
安全性, 包括用户帐户, 角色和权限。
存储, 包括表空间, 数据文件和回退段。
分布，包括数据库链接、Streams、高级队列以及高级复制的管理。
仓库，包括概要管理和 OLAP 管理。
工作空间, 一个或多个用户可以共享的, 用来对数据进行版本化更改的虚拟环境。
其他数据库功能, 取决于随您的数据库安装的产品。
Oracle Enterprise Manager 10g Console 简介
Oracle 建议使用 Enterprise Manager 10g Console 执行以下功能：
例程, 包括启动, 关闭和初始化。
方案, 包括表, 索引和所有其他方案对象。
安全性, 包括用户帐户, 角色和权限。
存储, 包括表空间, 数据文件和回退段。
使用 Enterprise Manager 10g Console 执行以下功能：
例程，包括启动、关闭和初始化例程。
数据库工具，包括备份管理、数据管理和对象分析。
诊断功能，包括数据库锁、顶级会话和顶级 SQL 的性能概览。
