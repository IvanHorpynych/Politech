Comparar Objetos de Base de Datos
Change Manager> Comparar Objetos de Base de Datos
El Asistente de Comparación de Objetos de Base de Datos permite comparar definiciones de objetos. Puede comparar definiciones de dos bases de datos, dos líneas base, una base de datos y una línea base o dos esquemas de una sola base de datos/línea base.

A continuación, puede utilizar el Asistente de Sincronización para modificar las definiciones de objetos de base de datos o esquema para que coincidan con sus correspondientes definiciones de objetos.
