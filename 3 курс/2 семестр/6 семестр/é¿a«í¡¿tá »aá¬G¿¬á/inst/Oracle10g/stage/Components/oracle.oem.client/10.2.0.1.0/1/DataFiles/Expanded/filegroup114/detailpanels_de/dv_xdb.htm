Verwaltung von XML Database
Oracle unterstützt XML Database, ein Hochleistungs-Objektsystem basierend auf dem XML-Datenmodell. XML Database speichert und ruft XML-Objekte über einen hierarchischen Pfadnamen ab. XML Database optimiert Zugriff und Aktualisierung von XML-Objekten, die in diesem System gespeichert sind. Außerdem wird die Zugriffskontrolle und Verwaltung der Konfiguration für diese Objekte ermöglicht.
Konfigurieren Sie XML Database zur Benutzung in Ihrer Umgebung.
Erstellen Sie eine Ressource in XML Database. Sie können die Zugriffskontrollberechtigungen für diese Ressource ändern, indem Sie sie im Navigatorbaum auswählen, mit der rechten Maustaste klicken und die Menüoption Berechtigungen für Benutzer/Rolle erteilen auswählen.
