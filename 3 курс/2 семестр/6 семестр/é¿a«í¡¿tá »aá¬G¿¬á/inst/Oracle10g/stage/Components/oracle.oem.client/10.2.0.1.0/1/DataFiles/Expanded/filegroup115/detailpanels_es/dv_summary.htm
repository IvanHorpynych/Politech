Gestión de Resúmenes
Oracle soporta vistas materializadas, a las que también se las conoce como resúmenes, para mejorar el rendimiento de un almacén de datos. Los resúmenes calculan previamente uniones y operaciones de agregación caras y las almacena en una tabla. Oracle acelera la ejecución de consultas mediante la reescritura transparente de consultas para que utilicen resúmenes. Las dimensiones ayudan a la operación de reescritura de consultas. Los logs de vistas materializadas realizan un seguimiento de los cambios realizados en la tabla maestra para realizar el refrescamiento incremental de vistas materializadas.
Utilice las Dimensiones para definir relaciones jerárquicas entre pares de columnas o juegos de columnas.
Utilice las Vistas Materializadas para mejorar el rendimiento del almacén de datos guardando uniones y operaciones de agregación calculadas previamente.
Utilice Log de Vista Materializada para realizar un seguimiento de los cambios en la tabla maestra para realizar el refrescamiento incremental de vistas materializadas.
Utilice Grupo de Refrescamiento para mantener las vistas materializadas consistentes unas con otras en lo relacionado con las transacciones.
Utilice el Asesor de Resúmenes para obtener recomendaciones para la creación de vistas materializadas para que la reescritura de consultas mejore el rendimiento de las aplicaciones de almacenes de datos.
