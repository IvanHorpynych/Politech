Editor de Planes
Change Manager> Editor de Planes
El Editor de Planes es un componente de Change Manager que se inicia al crear o editar un plan de cambios. Puede trabajar en la vista de árbol completa del Editor de Planes para definir cada elemento del plan de cambios:
  1. Crear solicitudes de cambio.
  2. Especificar bases de datos destino (en las que desea realizar los cambios).
  3. Generar el archivo de comandos.
  4. Ejecutar el archivo de comandos.
