DB Diff permet de comparer des définitions d'objet. Vous pouvez comparer les définitions de deux bases de données, de deux lignes de base, d'une base de données et d'une ligne de base, ou de deux schémas au sein d'une même base de données/ligne de base.

Vous pouvez ensuite utiliser l'assistant Synchronisation de DB Diff pour modifier des définitions d'objet de base de données ou de schéma en fonction des définitions d'objet correspondantes.
