Performance charts help you to monitor and assess how well your database is running. Charts that you can view include:
Overview, provides at-a-glance evaluation of the health of your database
TopSessions, allows you to monitor connected sessions and helps you to identify and correct performance problems
TopSQL, displays the most active SQL statements in your database
Locks, allows you to view and manage locks in your database
You can view additional realtime data using any of the many realtime charts available with Performance Manager. You can also record and view historical data with Capacity Planner. To run these tools, click the buttons below.
