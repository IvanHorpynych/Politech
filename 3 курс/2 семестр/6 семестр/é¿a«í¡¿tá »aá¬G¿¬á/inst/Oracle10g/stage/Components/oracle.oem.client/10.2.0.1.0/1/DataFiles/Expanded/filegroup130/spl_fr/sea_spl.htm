Rechercher des objets de base de données
Change Manager>Rechercher des objets de base de données
L'outil Rechercher des objets de base de données permet de rechercher des objets en indiquant les caractéristiques de leur nom.

Par exemple, vous pouvez rechercher toutes les tables ou les vues dont le nom contient la chaîne "EMP".
