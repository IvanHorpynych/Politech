XML Database Management
Oracle, provides support for XML Database, a high-performance object system based on XML data model. XML Database is designed to store and retrieve XML Objects via a hierarchical pathname. XML Database optimizes access and updates to XML objects stored in it. It also provides access control and configuration management for these objects.
Configure XML Database for use in your environment.
Create a resource in XML Database. You can modify access control privileges on this resource by selecting it in the navigation tree, doing a right mouse click and selecting Grant Privileges to User/Role menu item.
