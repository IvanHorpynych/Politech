스키마 관리
 
스키마 관리를 사용하여 다음 작업을 수행할 수 있습니다.
스키마 객체를 생성, 변경 및 삭제합니다.
스키마 객체에 대한 종속성을 표시합니다.
SQL 스크래치 패드를 사용하여 SQL 문을 실행합니다.
엑스포트 마법사를 사용하여 데이터를 이진 형식 엑스포트 파일로 엑스포트하고 임포트 마법사를 사용하여 이 파일에서 데이터를 임포트합니다.
분석 마법사를 사용하여 데이터베이스 객체를 분석합니다.
목록에서 여러 객체를 선택하여 해당 객체에서 작업합니다.
데이터베이스 객체를 찾습니다.
객체에 대한 DDL 스크립트를 생성합니다.
참고 Oracle은 관리 및 진단에 Enterprise Manager 10g Console을 사용할 것을 권장합니다.
