Replicación de Varios Maestros
 
Replicación de Varios Maestros permite que varias ubicaciones (denominadas Ubicaciones Maestras), actuando como peers iguales, gestionen grupos de objetos de base de datos replicados.
Para configurar la replicación de varios maestros:
  1. Haga clic en Configurar Ubicaciones Maestras... para iniciar un asistente que permite configurar un entorno de replicación de varios maestros.

  2. Haga clic en Crear Grupo Maestro... para crear un grupo maestro que gestione los objetos replicados. Al terminar, el nuevo grupo aparecerá en el navegador y se ejecutará una solicitud para iniciar el proceso de replicación.

  3. Desde el navegador, seleccione el grupo maestro que acaba de crear. La replicación de varios maestros se ejecuta si el estado del grupo maestro es Iniciado.
Para obtener más información sobre la replicación de varios maestros, haga clic en el botón Ayuda.
