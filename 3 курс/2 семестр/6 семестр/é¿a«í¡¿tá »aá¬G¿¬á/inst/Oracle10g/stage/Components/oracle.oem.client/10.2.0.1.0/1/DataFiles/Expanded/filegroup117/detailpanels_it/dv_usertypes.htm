Tipi utente
Oracle, come sistema di gestione di database relazionali a oggetti, consente di definire tipi di dati aggiuntivi e di utilizzarli nel modello relazionale.
Utilizzare Object type per creare un tipo complesso che costituisca un'astrazione di entità reali gestite e utilizzate dai programmi applicativi.
Utilizzare Tipo di array per creare un set ordinato di elementi di dati dello stesso tipo.
Utilizzare Tipo di tabella per creare un set non ordinato di elementi di dati dello stesso tipo di dati.
Utilizzare Schema XML per creare tipi complessi ed elementi di memorizzazione per i documenti XML basati sullo schema XML. Questo object type è disponibile per Oracle versione 9.2 e versioni successive.
SUGGERIMENTO Per le operazioni di amministrazione e diagnostica, la Oracle consiglia di utilizzare Enterprise Manager 10g Console.
