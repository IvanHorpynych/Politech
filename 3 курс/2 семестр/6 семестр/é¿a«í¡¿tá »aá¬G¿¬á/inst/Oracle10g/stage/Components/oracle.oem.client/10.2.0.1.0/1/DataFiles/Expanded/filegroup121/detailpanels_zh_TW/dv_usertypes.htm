User Types
Oracle, as an object-relational database management system, allows you to define additional types of data and to use them within the relational model.
Use Object Type to create a complex type that is an abstraction of real-world entities that your application program deals with.
Use Array Type to create an ordered set of data elements of the same type.
Use Table Type to create an un-ordered set of data elements of the same datatype.
Use XML Schema to create complex types and storage elements for XML documents based on the XML schema. This object type is available for Oracle version 9.2 and above.
TIP Oracle recommends using the Enterprise Manager 10i Console for administration and diagnostics.
