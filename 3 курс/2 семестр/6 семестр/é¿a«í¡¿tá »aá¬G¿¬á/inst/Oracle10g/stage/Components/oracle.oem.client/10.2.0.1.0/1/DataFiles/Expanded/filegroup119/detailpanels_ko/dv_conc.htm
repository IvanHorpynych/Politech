동시 사용 관리자
 
동시 사용 관리자는 Oracle Applications 데이터베이스에 대한 요청 처리 일정을 잡습니다. 콘솔을 사용하여 다음 작업을 수행할 수 있습니다.
개입해야 하는 조건에 대해 동시 사용 관리자를 모니터하는 이벤트를 생성합니다.
이러한 조건을 수정하기 위한 일괄 처리 작업 일정을 잡습니다.
통지 일시 중단 기간 일정을 잡거나 해당 동시 사용 관리자에 대해 모든 통지를 일시 중단합니다.
용량 계획자를 사용하여 기록 추세를 분석하고 이후 용량을 예상합니다.
성능 관리자 차트를 사용하여 동시 사용 관리자 통계를 분석합니다.
동시 사용 관리자, 동시 사용 프로그램 및 동시 사용 프로그램 프로세싱에서 기록 데이터의 보고서를 생성합니다.
네비게이터 트리에서 동시 사용 관리자를 삭제합니다.
