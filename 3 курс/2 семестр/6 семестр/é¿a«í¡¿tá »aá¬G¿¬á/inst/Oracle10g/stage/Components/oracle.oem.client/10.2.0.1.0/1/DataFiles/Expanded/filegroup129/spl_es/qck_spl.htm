Cambio Rápido de la Base de Datos
Change Manager> Cambio Rápido de la Base de Datos
Cambio Rápido de la Base de Datos permite realizar cambios en una sola definición de objeto en una base de datos. Cambio Rápido de la Base de Datos crea el plan de cambios adecuado para realizar los cambios.

al contrario que otras herramientas del paquete Oracle Change Management, Cambio Rápido de la Base de Datos no guarda automáticamente el trabajo como plan de cambios, pero permite hacerlo.
