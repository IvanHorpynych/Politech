Einsatzvorlagen
Materialized View-Replikation>Einsatzvorlagen
Mit Einsatzvorlagen können Sie eine Sammlung von Materialized View-Definitionen auf einem Master-Rechner definieren und Parameter in den Definitionen verwenden, damit die Materalized Views für einzelne Benutzer angepasst werden können.
So erstellen Sie eine Einsatzvorlage auf einem Master-Rechner:
  1. Klicken Sie auf Master-Rechner einrichten..., um den Master-Rechner einzurichten, falls dies noch nicht geschehen ist.
  2. Wenn Sie aktualisierbare Materialized Views in der Vorlage aufnehmen möchten, müssen deren Basistabellen in der Master-Gruppe vorhanden sein.
    Klicken Sie auf Master-Gruppe erstellen..., um eine Master-Gruppe zu erstellen, die die replizierten Objekte verwaltet.
  3. Klicken Sie auf Vorlage erstellen..., um den Assistenten zu starten, mit dem Sie die Einsatzvorlage erstellen können.
So passen Sie die Vorlage für einzelne Benutzer an und wenden sie auf Materialized View-Rechnern an:
  1. Klicken Sie auf Generierung von Vorlagen-Skript..., um die Generierung von Offline-Vorlagen-Skripts abzuschließen.
  2. Verteilen Sie die Instanziierungsdateien an jeden Benutzer. Die Benutzer können das Skript auf dem Materialized View-Rechner mit SQL*Plus ausführen, um die Materialized View-Umgebung zu erstellen.
Klicken Sie auf die Schaltfläche Hilfe, wenn Sie mehr über Einsatzvorlagen erfahren möchten.
