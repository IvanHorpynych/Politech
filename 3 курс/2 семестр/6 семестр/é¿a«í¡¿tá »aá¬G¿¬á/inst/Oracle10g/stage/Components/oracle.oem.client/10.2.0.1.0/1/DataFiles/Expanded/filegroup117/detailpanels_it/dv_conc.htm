Concurrent Manager
 
I Concurrent Manager pianificano le richieste di elaborazione sui database delle applicazioni Oracle. Utilizzando Console, è possibile:
Creare eventi per monitorare i Concurrent Manager e individuare le condizioni che richiedono un intervento.
Pianificare processi batch per correggere tali condizioni.
Pianificare periodi di sospensione delle notifiche oppure sospendere tutte le notifiche per un Concurrent Manager specifico.
Utilizzare Capacity Planner per analizzare le tendenze cronologiche e la capacità futura dei progetti.
Utilizzare i grafici di Perfomance Manager per analizzare le statistiche dei Concurrent Manager.
Generare report dei dati cronologici forniti dai Concurrent Manager, dai programmi concorrenti e dall'elaborazione di progarmmi concorrenti
Eliminare un Concurrent Manager dalla struttura del Navigator.
