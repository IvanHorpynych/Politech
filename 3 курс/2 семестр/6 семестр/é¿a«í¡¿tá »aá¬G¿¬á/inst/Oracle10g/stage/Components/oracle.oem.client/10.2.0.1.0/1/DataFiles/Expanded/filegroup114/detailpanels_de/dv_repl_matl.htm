Materialized View-Replikation
 
Mit der Materialized View-Replikation können Tabellen in der Quelldatenbank (die als Master-Rechner bezeichnet wird) vollständig oder teilweise auf einer oder mehreren Zieldatenbanken (die als Materialized View-Rechner bezeichnet werden) repliziert werden.
Es gibt verschiedene Möglichkeiten zur Ausführung der Materialized View-Replikation. Klicken Sie auf einen Link für weitere Informationen:
  • Verwenden Sie Einsatzvorlagen, um eine weit verteilte Materialized View-Umgebung effizient einzusetzen und zu verwalten.
  • Verwenden Sie Materialized Views, um eine schreibgeschützte Materialized View-Umgebung anzuwenden und zu verwalten.
  • Verwenden Sie Materialized View-Gruppen, um eine aktualisierbare Materialized View-Umgebung anzuwenden und zu verwalten.
  • Verwenden Sie Aktualisierengruppen, um mehrere Materialized Views in einem Schritt zu aktualisieren.
Klicken Sie auf die Schaltfläche Hilfe, wenn Sie mehr über die Materialized View-Replikation erfahren möchten.
