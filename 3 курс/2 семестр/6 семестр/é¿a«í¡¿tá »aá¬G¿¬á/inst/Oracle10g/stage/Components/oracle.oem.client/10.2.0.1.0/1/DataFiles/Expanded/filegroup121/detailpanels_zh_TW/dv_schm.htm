Schema Management
 
Use Schema Management to:
Create, alter and drop schema objects.
Show dependencies on schema objects.
Use SQL Scratchpad to execute SQL statements.
Use Export Wizard to export the data to a binary-format export file and use the Import Wizard to import data from this file.
Use Analyze Wizard to analyze database objects.
Work on multiple objects by selecting them from the list.
Find database objects.
Generate the DDL script for an object.
TIP Oracle recommends using the Enterprise Manager 10i Console for administration and diagnostics.
