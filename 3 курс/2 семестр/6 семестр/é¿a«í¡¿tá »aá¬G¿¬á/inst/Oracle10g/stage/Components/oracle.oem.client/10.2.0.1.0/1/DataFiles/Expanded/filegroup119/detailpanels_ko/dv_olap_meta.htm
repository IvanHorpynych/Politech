OLAP 메타 데이터 객체
OLAP 관리 > OLAP 메타 데이터 객체
분석 프로세싱을 수행하려면 데이터 웨어하우스가 OLAP 메타 데이터 객체와 함께 미리 구성되어 있어야 하며 구체화된 뷰를 생성하여 데이터 웨어하우스에 대한 액세스가 최적화되어 있어야 합니다. OLAP 메타 데이터 객체는 다음과 같습니다.



  • 차원 객체 - 데이터 웨어하우스 차원 테이블을 레벨, 속성 및 계층으로 구성합니다. 이러한 객체를 생성하려면 차원 마법사를 실행합니다.



  • 큐브 - 다중 차원 데이터의 논리적 표현이며 적어도 하나의 차원을 포함해야 합니다. 큐브를 생성하려면 큐브 마법사를 실행합니다.



  • 측정 단위 폴더 - 측정 단위를 관리 가능한 그룹으로 구성합니다. 측정 단위 폴더를 생성하려면 측정 단위 폴더 대화상자를 실행합니다.

네비게이터 트리에서 큐브를 선택한 경우 해당 큐브를 마우스 오른쪽 단추로 눌러 다음 툴을 실행합니다.

  • 요약 권고자 마법사 - 이 마법사는 큐브에 의해 정의된 데이터에 액세스하는 OLAP 클라이언트 응용 프로그램의 성능을 최적화하는 구체적인 뷰를 권장합니다.



  • Cube Viewer - 이 응용 프로그램은 큐브에 의해 나타나는 다중 차원 데이터를 표시합니다.
