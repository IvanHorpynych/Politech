Gestione memoria
 
Utilizzare Gestione memoria per effettuare le operazioni riportate di seguito.
Creare oggetti di memorizzazione quali Tablespace, Datafile, Segmenti di rollback e Gruppi di redo log.
Aggiungere datafile e segmenti di rollback a una tablespace.
Eliminare oggetti di memorizzazione.
Mettere gli oggetti in linea o fuori linea.
Mostrare le dipendenze dagli oggetti.
Eseguire operazioni di backup, recupero e gestione sui file del database.
Visualizzare il layout memoria. Questa funzione è disponibile per Oracle versione 9.2 e versioni successive.
SUGGERIMENTO Per le operazioni di amministrazione e diagnostica, la Oracle consiglia di utilizzare Enterprise Manager 10g Console.
