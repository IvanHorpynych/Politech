Gestión de OLAP
Para realizar un procesamiento analítico, el almacén de datos debe estar preconfigurado con objetos de metadatos de OLAP, que incluyen:
Dimensión: Organiza tablas de dimensiones de almacenes de datos en niveles, atributos y jerarquías.
Cubos: Son representaciones lógicas de datos multidimensionales y deben incluir al menos una dimensión.
Carpetas de Medidas: Organizan las medidas en grupos que se pueden gestionar.
Para crear vistas materializadas para optimizar el almacén de datos, utilice la siguiente aplicación:
Haga clic con el botón derecho del mouse en un nodo de cubo e inicie el Asistente de Asesor de Resúmenes de OLAP. El asistente recomienda vistas materializadas que optimizan el rendimiento de las aplicaciones cliente de OLAP que acceden a los datos definidos por un cubo. Este asistente está disponible en Oracle versión 9.2 y superiores.
Hay otras dos aplicaciones disponibles para la gestión de OLAP.
Haga clic con el botón derecho del mouse en un nodo de cubo e inicie Cube Viewer para visualizar los datos multidimensionales representados por el cubo.
Haga clic con el botón derecho del mouse en la carpeta de OLAP para iniciar Hoja de Trabajo de OLAP, que se utiliza para codificar y editar el código 4GL de OLAP. Esta aplicación está disponible en Oracle versión 9.2 y superiores.
