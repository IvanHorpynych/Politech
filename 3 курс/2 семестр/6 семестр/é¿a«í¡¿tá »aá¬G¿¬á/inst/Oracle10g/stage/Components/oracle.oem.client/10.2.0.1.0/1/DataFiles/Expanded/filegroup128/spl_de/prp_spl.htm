Mit DB Propagate können Sie Änderungspläne erstellen und implementieren, mit denen Objektdefinitionen angegeben werden, die Sie an anderer Stelle reproduzieren möchten.

Sie können neue Änderungspläne erstellen oder Pläne bearbeiten, die vorher mit DB Propagate erstellt wurden. Sie können die Änderungen für eine oder mehrere Datenbanken bereitstellen.
