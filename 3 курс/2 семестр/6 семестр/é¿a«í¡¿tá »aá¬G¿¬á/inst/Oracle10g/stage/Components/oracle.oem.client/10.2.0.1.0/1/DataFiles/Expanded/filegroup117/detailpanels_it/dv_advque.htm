Code avanzate
 
Enterprise Manager fornisce strumenti di facile utilizzo per l'amministrazione delle code avanzate. È possibile:
Creare, modificare ed eliminare tabelle di coda.
Creare, eliminare, avviare e interrompere code.
Aggiungere e rimuovere destinatari predefiniti per le code.
Aggiungere, abilitare, disabilitare ed eliminare pianificazioni di code per la propagazione dei messaggi.
Visualizzare la topologia, le statistiche e gli errori delle pianificazioni a livello di database oltre che a livello di coda.

Per ulteriori informazioni sulle code avanzate,
fare clic sul pulsante ?.
