Gestione di database distribuiti
Oracle supporta le tecnologie di replica dei dati e di messaggistica per l'utilizzo di applicazioni distribuite e di sistemi di database distribuiti.
Utilizzare la cartella Transazioni equivoche per visualizzare le transazioni di commit a due fasi non riuscite.
Utilizzare Database link per creare, modificare ed eliminare i link tra i database.
Utilizzare la cartella Streams per replicare i dati in una rete di database. È possibile monitorare e amministrare l'ambiente Streams. Questa cartella è disponibile per Oracle versione 9.2 e versioni successive.
Utilizzare la cartella Code avanzate per integrare applicazioni mediante la tecnologia di messaggistica. È possibile monitorare e amministrare l'ambiente Code avanzate.
Utilizzare la cartella Advanced Replication per replicare i dati in un gruppo di database (replica con più siti principali) o per replicare uno snapshot di dati in più database (replica snapshot). È possibile monitorare e amministrare l'ambiente Advanced Replication.
