Gestión de Bases de Datos XML
Oracle proporciona soporte para bases de datos XML, un sistema de objetos de alto rendimiento basado en el modelo de datos XML. La base de datos XML está diseñada para almacenar y recuperar objetos XML a través de un nombre de ruta de acceso jerárquica. Esta base de datos optimiza el acceso y actualiza a los objetos XML almacenados en ella. Proporciona además control de acceso y gestión de configuración para estos objetos.
Configure la base de datos XML para utilizarla en su entorno.
Cree un recurso en la base de datos XML. Puede modificar los privilegios de control de acceso en este recurso seleccionándolo en el árbol de navegación, haciendo clic con el botón derecho del mouse y seleccionando la opción de menú Otorgar Privilegios a Usuario o Rol.
