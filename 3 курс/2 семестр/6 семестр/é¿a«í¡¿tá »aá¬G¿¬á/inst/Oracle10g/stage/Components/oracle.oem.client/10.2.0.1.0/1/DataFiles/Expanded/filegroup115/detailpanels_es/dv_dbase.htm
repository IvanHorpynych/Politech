Bases de Datos
 
La carpeta de bases de datos contiene todas las bases de datos a las que se tiene acceso. Puede ampliar una base de datos concreta para administrar cualquiera de los siguientes elementos:
Instancias, incluida la inicialización.
Esquemas, incluidas las tablas, los índices y todos los demás objetos de esquema.
Seguridad, incluidas las cuentas de usuario, los roles y los privilegios.
Almacenamiento, incluidos los tablespaces, los archivos de datos y los segmentos de rollback.
Distribuido, incluida la gestión de Enlaces de Base de Datos, Streams, Colas Avanzadas y Replicación Avanzada.
Almacén, incluida la gestión de resúmenes y la gestión de OLAP.
Espacios de Trabajo, un entorno virtual que pueden compartir uno o más usuarios para realizar cambios versionados en los datos.
Otras funciones de la base de datos, según los productos instalados con la base de datos.
Introducción a la Consola de Oracle Enterprise Manager 10g
Oracle recomienda utilizar la Consola de Enterprise Manager 10g para las siguientes funciones:
Instancias, incluido el inicio, el cierre y la inicialización.
Esquemas, incluidas las tablas, los índices y todos los demás objetos de esquema.
Seguridad, incluidas las cuentas de usuario, los roles y los privilegios.
Almacenamiento, incluidos los tablespaces, los archivos de datos y los segmentos de rollback.
Utilice la Consola de Enterprise Manager 10g para las siguientes funciones:
Instancias, incluido el inicio, el cierre y la inicialización.
Herramientas de Base de Datos, incluida la gestión de copias de seguridad, la gestión de datos y el análisis de objetos.
Funciones de Diagnóstico, incluida la visión general de rendimiento de los bloqueos de base de datos, las sesiones principales y el SQL principal.
