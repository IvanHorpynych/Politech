Konfiguration von OLAP-Diensten
OLAP Management > Konfiguration von OLAP-Diensten
OLAP-Dienste sind Datenbankprozesse, die analytische Informationen verarbeiten und gleichzeitig das Load-Sharing ermöglichen. Sie können mit dem OLAP Services Instance Manager verwaltet und konfiguriert werden. Mit diesem Werkzeug können Sie:
 
OLAP-Dienste starten, stoppen und unterbrechen.
neue Dienste erstellen und bestehende Dienste entfernen.
Konfigurationsparameter ändern.
Client-Sessions überwachen.
System-Logs anzeigen.
 
OLAP Services Instance Manager jetzt starten?
