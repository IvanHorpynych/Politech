Objets de métadonnées OLAP
OLAP Management > Objets de métadonnées OLAP
Pour effectuer un traitement analytique, la base de données Data Warehouse doit être préconfigurée avec des objets de métadonnées OLAP et vous devez optimiser l'accès à cette base de données en créant des vues matérialisées. Les objets de métadonnées OLAP sont les suivants :

  • Les dimensions organisent les tables de dimensions de la base de données Data Warehouse en niveaux, attributs et hiérarchies. Pour créer ces objets, lancez l'assistant Dimension.

  • Les cubes sont des représentations logiques des données multidimensionnelles et doivent contenir au moins une dimension. Pour créer des cubes, lancez l'assistant Cube.

  • Les dossiers d'indicateurs qui organisent les indicateurs en groupes gérables. Pour créer des dossiers d'indicateurs, lancez la boîte de dialogue Dossier d'indicateurs.
Lorsqu'un cube est sélectionné dans l'arborescence du Navigateur, cliquez dessus avec le bouton droit de la souris pour lancer les outils suivants :
  • Assistant Summary Advisor : cet assistant recommande l'utilisation de vues matérialisées optimisant les performances des applications client OLAP qui accèdent aux données définies par un cube.

  • Cube Viewer : cette application affiche les données multidimensionnelles représentées par le cube.
