Mit dem Assistenten Baseline erstellen können Sie die Definitionen von Objekten zu einem bestimmten Zeitpunkt erfassen. Die erfassten Definitionen werden als Baselines in einem internen Format gespeichert, das von Change Management Pack-Anwendungen benutzt wird.

Mit dem Assistenten Baseline erstellen können Sie auch ein Reverse-Engineering von Objekt- oder Schemadefinitionen in DDL (Data Definition Language) vornehmen.
