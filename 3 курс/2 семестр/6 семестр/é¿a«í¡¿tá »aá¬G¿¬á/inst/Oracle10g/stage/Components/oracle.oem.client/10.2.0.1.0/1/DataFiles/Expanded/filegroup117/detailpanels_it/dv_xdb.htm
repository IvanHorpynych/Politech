Gestione del database XML
Oracle fornisce il supporto per il database XML, un sistema di oggetti dalle alte prestazioni basato sul modello di dati XML. Il database XML è stato progettato per la memorizzazione e il recupero degli oggetti XML tramite un percorso gerarchico e ottimizza l'accesso e gli aggiornamenti relativi agli oggetti XML che vi sono memorizzati. Il database XML fornisce inoltre funzioni di controllo dell'accesso e di gestione della configurazione per questi oggetti.
Configurare il database XML per utilizzarlo nel proprio ambiente di lavoro.
Creare una risorsa nel database XML. È possibile modificare i privilegi di controllo dell'accesso sulla risorsa creata selezionandola nella struttura del Navigator, facendo clic con il pulsante destro del mouse e selezionando il comando di menu Aggiungi privilegi agli utenti/ai ruoli.
