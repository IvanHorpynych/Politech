Directory Servers
 
The Directory Servers folder lists all the Directory servers that have been discovered by the Console, and displays their properties. Right-click any Directory server to:
Startup or shutdown the Directory server.
Ping the server to verify that it is running.
View server properties, such as server version.
Delete the Directory server from the Navigator tree.
Schedule notification blackout periods, or black out all notifications for that server.
You can also create events to monitor Directory servers for conditions requiring intervention, and schedule jobs to correct those conditions.
