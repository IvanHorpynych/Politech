OLAP Management
Pour effectuer un traitement analytique, la base de données data warehouse doit être préconfigurée avec des objets de métadonnées OLAP, à savoir :
Une dimension, qui organise les tables de dimensions de la base de données data warehouse en niveaux, attributs et hiérarchies.
Des cubes, qui sont des représentations logiques des données multidimensionnelles et qui doivent contenir au moins une dimension.
Des dossiers d'indicateurs, qui organisent les indicateurs en groupes gérables.
Pour créer des vues matérialisées destinées à l'optimisation de la base de données data warehouse, utilisez l'application suivante :
Cliquez avec le bouton droit de la souris sur le noeud d'un cube et lancez l'assistant OLAP Summary Advisor, qui recommande l'utilisation de vues matérialisées optimisant les performances des applications client OLAP qui accèdent aux données définies par un cube. Cet assistant est disponible pour Oracle versions 9.2 et ultérieures.
Deux applications supplémentaires sont disponibles pour la gestion OLAP.
Cliquez avec le bouton droit de la souris sur le noeud d'un cube, puis lancez Cube Viewer pour afficher les données multidimensionnelles représentées par ce cube.
Cliquez avec le bouton droit de la souris sur un dossier OLAP pour lancer OLAP Worksheet, utilisé pour le codage et la modification du code 4GL OLAP. Cette application est disponible pour Oracle versions 9.2 et supérieures.
