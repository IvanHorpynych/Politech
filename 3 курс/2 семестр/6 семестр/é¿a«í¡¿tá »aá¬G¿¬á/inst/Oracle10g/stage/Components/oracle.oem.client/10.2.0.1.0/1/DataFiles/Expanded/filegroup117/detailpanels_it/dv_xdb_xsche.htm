Oggetti basati sullo schema XML
Gestione del database XML>Oggetti basati sullo schema XML
Lo schema XML è un linguaggio di definizione di schemi, anch'esso in XML, che può essere utilizzato per descrivere la struttura e la semantica di documenti dell'istanza XML conformi.
In Oracle 9.2 è possibile effettuare le operazioni riportate di seguito.
Registrare uno schema XML con il database, utilizzando una risorsa esistente del database XML oppure utilizzando un file dello schema del database XML accessibile localmente o via Internet.
Creare una memoria strutturata basata sullo schema XML.
Questa operazione può essere eseguita in uno dei due seguenti modi:
Creare una tabella basata sulla struttura definita nello schema XML.
Creare una tabella con colonne basata sulla struttura definita nello schema XML.
La creazione di una memoria basata sullo schema XML offre i seguenti vantaggi:
  • i documenti dell'istanza XML che vi sono memorizzati occupano molto meno spazio;
  • l'accesso a porzioni di dati particolari, specificate tramite XPath, presenta un alto livello di ottimizzazione;
  • le operazioni di aggiornamento su parti del documento dell'istanza XML risultano notevolmente migliorate.
Sottoporre a wrapping i dati esistenti nei formati XML creando viste basate sullo schema XML.
Utilizzando espressioni Xpath, è inoltre possibile creare indici funzionali sulle tabelle basate sullo schema XML.
