Replication Management
 
La réplication est un processus consistant à copier et à mettre à jour des objets de base de données, tels que des tables, dans plusieurs bases de données qui forment un système de bases de données distribuées.
Utilisez la console pour effectuer les opérations suivantes :
Réplication multimaître : sélectionnez l'option correspondante dans le Navigateur.
Réplication de vues matérialisées : sélectionnez l'option correspondante dans le Navigateur.
