Plan Editor
 
The Plan Editor is a component of Change Manager, which launches when you create or edit a change plan. You then work from top to bottom in the Plan Editor tree view to define each element of the change plan:
  1. Create change requests.
  2. Specify destination databases (where you want to make the changes).
  3. Generate your script.
  4. Execute the script.

For more information, see the Quick Tour.
