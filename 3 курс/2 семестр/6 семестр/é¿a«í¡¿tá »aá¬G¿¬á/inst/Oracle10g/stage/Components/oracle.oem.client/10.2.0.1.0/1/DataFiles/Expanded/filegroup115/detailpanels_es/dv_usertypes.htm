Tipos de Usuario
Oracle, como sistema de gestión de bases de datos relacionadas con objetos, permite definir más tipos de dato y utilizarlos dentro del modelo relacional.
Utilice Tipo de Objeto para crear un tipo complejo que sea una abstracción de entidades del mundo real con las que trabaja su aplicación.
Utilice Tipo de Matriz para crear un juego ordenado de elementos de datos del mismo tipo.
Utilice Tipo de Tabla para crear un juego sin ordenar de elementos de datos del mismo tipo de dato.
Utilice Esquema XML para crear tipos complejos y elementos de almacenamiento para documentos XML basados en el esquema XML. Este tipo de objeto está disponible en Oracle versión 9.2 y superiores.
CONSEJO Oracle recomienda utilizar la Consola de Enterprise Manager 10g para llevar a cabo tareas de administración y de diagnóstico.
