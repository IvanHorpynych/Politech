源类型
Oracle 支持使用 SQL, PL/SQL, Java 或 C 编写的代码部件对象, 用来开发数据库中的应用程序逻辑。
使用程序包可创建或浏览存储为部件的一组相关过程和函数。
使用程序包体可创建或浏览程序包的所有公用和专用构造。您可以编译和调试程序包体。
使用过程可通过 SQL 和 PL/SQL 来开发代码。
使用函数可开发始终向调用程序返回单个值的代码。
使用触发器可开发在修改表或视图, 或者发生某些用户操作或数据库系统操作时执行的代码。
使用 Java 源可将 Java 代码编译为一组标准的, 平台无关的字节码, 这些字节码可与 Java 虚拟机 (JVM) 交互。
将 Java 类加载到数据库中, 并使用数据库中的 Java 虚拟机 (JVM) 执行。
提示 Oracle 建议使用 Enterprise Manager 10g Console 执行管理和诊断任务。
