Directory Server
 
Im Ordner Directory Server werden alle Directory Server aufgeführt, die von der Konsole erkannt wurden, und werden deren Attribute angezeigt. Klicken Sie mit der rechten Maustaste auf einen beliebigen Directory Server, um:
Den Directory Server hoch- oder herunterzufahren.
Einen Ping-Befehl für den Server auszuführen, um sicherzustellen, dass er läuft.
Server-Attribute anzuzeigen, wie z.B. Server-Version.
Den Directory Server aus dem Navigatorbaum zu löschen.
Zeiträume für Benachrichtigungs-Blackouts festlegen oder einen Blackout für alle Benachrichtigungen für diesen Server bestimmen.
Sie können auch Ereignisse erstellen, um Directory Server auf Bedingungen zu überwachen, die einen Eingriff erfordern, und Jobs zur Behebung dieser Bedingungen planen.
