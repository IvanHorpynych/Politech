Créer une ligne de base
Change Manager>Créer une ligne de base
L'assistant Créer une ligne de base permet de capturer les définitions d'objet à un moment donné. Les définitions capturées sont stockées en tant que lignes de base, sous un format interne utilisé par les applications Change Management Pack.

Vous pouvez également utiliser l'assistant Créer une ligne de base pour reconstituer la logique de définitions d'objet ou de schéma en langage DDL.
