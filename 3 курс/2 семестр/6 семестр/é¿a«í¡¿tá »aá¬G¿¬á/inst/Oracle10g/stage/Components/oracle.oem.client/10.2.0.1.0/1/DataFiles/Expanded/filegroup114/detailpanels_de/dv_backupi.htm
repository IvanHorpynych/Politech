Backup und Wiederherstellung
Instance Management>Backup und Wiederherstellung
Backup-, Zurückschreiben- und Recovery-Vorgänge mit Datenbankdateien, wie z.B. Datendateien, archivierten Logs, Kontrolldatei und Server Parameter File können mit dem Oracle Recovery Manager-(RMAN-)Dienstprogramm ausgeführt werden. Die Konsole bietet eine benutzerfreundliche Schnittstelle zu dem RMAN-Dienstprogramm zur Ausführung dieser Aufgaben.
Verwenden Sie den Backup-Assistenten zur Sicherung von Datenbankobjekten.
Verwenden Sie den Wiederherstellungs-Assistenten zur Wiederherstellung von Datenbankobjekten.
Verwenden Sie den Verwaltungs-Assistenten zur Einrichtung von Backup- und Erhaltungs-Policys in einer Zieldatenbank oder zur Registrierung, Rücksetzung oder Resynchronisierung der Zieldatenbanken mit dem Wiederherstellungskatalog.
Leiten Sie RMAN-Skript-Job weiter, indem Sie Ihre eigenen RMAN-Skripts verwenden, um RMAN-Befehle und RMAN-Vorgänge auszuführen, einschließlich Backup und Wiederherstellung.
