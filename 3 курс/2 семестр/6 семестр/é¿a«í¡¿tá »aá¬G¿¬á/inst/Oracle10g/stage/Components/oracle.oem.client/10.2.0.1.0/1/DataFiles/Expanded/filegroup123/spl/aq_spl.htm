Enterprise Manager provides easy administration of Advanced Queues. You can:
Create, alter and drop queue tables.
Create, drop, start and stop queues.
Add and remove subscribers for queues.
Add, enable, disable and drop schedules of queues for message propagation.
View topology, statistics and errors for schedules at database level as well as queue level.
Browse messages for queues and queue tables.
To learn more about Advanced Queues using DBA Studio, click the Help button.
