Plantillas de Despliegue
Replicación de Vista Materializada>Plantillas de Despliegue
Las plantillas de despliegue permiten definir una recopilación de definiciones de vistas materializadas en una ubicación maestra y utilizar parámetros en las definiciones para poder personalizar las vistas materializadas para cada usuario.
Para crear una plantilla de despliegue en una ubicación maestra:
  1. Haga clic en Configurar Ubicaciones Maestras... para configurar la ubicación maestra, si aún no se ha realizado.
  2. Si desea incluir vistas materializadas actualizables en la plantilla, sus tablas base deben estar en el grupo maestro.
    Haga clic en Crear Grupo Maestro... para crear un grupo maestro que gestione los objetos replicados.
  3. Haga clic en Crear Plantilla... para iniciar el asistente que le ayude a crear la plantilla de despliegue.
Para personalizar la plantilla para cada usuario y desplegarla en ubicaciones de vistas materializadas:
  1. Haga clic en Generación de Archivos de Comandos de Plantilla... para completar la generación de archivos de comandos de plantilla fuera de línea.
  2. Distribuya archivos de instanciación a cada usuario. Los usuarios pueden ejecutar el archivo de comandos en la ubicación de vistas materializadas mediante SQL*Plus para crear el entorno de vistas materializadas.
Para obtener más información sobre las plantillas de despliegue, haga clic en el botón Ayuda.
