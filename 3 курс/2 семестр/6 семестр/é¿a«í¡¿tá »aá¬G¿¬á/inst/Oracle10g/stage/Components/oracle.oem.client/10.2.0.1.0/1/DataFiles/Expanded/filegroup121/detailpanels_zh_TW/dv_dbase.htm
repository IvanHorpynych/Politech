Databases
 
The databases folder contains all databases that are accessible to you. You can expand a specific database to administer any of the following:
Instances, including initialization.
Schemas, including tables, indexes, and all other schema objects.
Security, including user accounts, roles, and privileges.
Storage, including tablespaces, datafiles, and rollback segments.
Distributed, including management of Database Links, Streams, Advanced Queues, and Advanced Replication.
Warehouse, including summary management and management of OLAP.
Workspaces, a virtual environment that one or more users can share to make versioned changes to data.
Other database features, depending on the products installed with your database.
Oracle Enterprise Manager 10i Console Introduction
Oracle recommends using the Enterprise Manager 10i Console for following features:
Instances, including startup, shutdown, and initialization.
Schemas, including tables, indexes, and all other schema objects.
Security, including user accounts, roles, and privileges.
Storage, including tablespaces, datafiles, and rollback segments.
Use the Enterprise Manager 10i Console for following features:
Instances, including starup, shutdown, and initialization.
Database Tools, including backup management, data management, and object analyze.
Diagnostics Features, including performance overview for database locks, top sessions, and top sql.
