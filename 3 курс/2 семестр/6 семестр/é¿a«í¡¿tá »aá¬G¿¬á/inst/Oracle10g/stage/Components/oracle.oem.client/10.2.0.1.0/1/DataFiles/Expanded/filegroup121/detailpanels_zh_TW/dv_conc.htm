Concurrent Managers
 
Concurrent Managers schedule processing requests on Oracle Applications databases. With the Console, you can:
Create events to monitor Concurrent Managers for conditions requiring intervention.
Schedule batch jobs to correct those conditions.
Schedule notification blackout periods, or black out all notifications for that Concurrent Manager.
Use Capacity Planner to analyze historical trends and project future capacity.
Use Perfomance Manager charts to analyze Concurrent Manager statistics.
Generate reports of historical data from Concurrent Managers, Concurrent Programs, and Concurrent Program Processing
Delete a Concurrent Manager from the Navigator tree.

To learn more about administration of Concurrent Managers, click the Quick Tour button.
