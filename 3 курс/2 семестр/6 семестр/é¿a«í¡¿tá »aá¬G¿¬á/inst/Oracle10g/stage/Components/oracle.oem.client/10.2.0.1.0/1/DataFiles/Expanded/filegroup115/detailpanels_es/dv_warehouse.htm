Gestión de Almacenes de Datos
 
Un almacén de datos es una base de datos relacional diseñada para realizar consultas y análisis en lugar de procesamientos de transacciones. Oracle soporta el análisis completo de datos de almacenes de datos mediante aplicaciones de procesamiento analítico en línea (OLAP). Oracle también proporciona mecanismos para mejorar el rendimiento del almacén de datos mediante el uso de resúmenes.
 
Seleccione la carpeta Gestión de Resúmenes del navegador para obtener información sobre la administración de resúmenes.
