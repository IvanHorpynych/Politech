Change Manager
 
Change Manager et ses outils intégrés permettent d'effectuer les opérations suivantes :
Capture des définitions d'objet de base de données dans une "ligne de base".
Comparaison et synchronisation des définitions d'objet de base de données.
Recherche d'objets par nom.
Modification de définitions d'objet de base de données dans des bases de données.
Cliquez sur les icônes situées sur la droite pour obtenir une brève description de chaque outil ou reportez-vous à Sélection d'un outil de modification.
