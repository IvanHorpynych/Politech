Gestion d'Oracle XML DB
Oracle offre la prise en charge de XML DB, système d'objets hautes performances basé sur un modèle de données XML. Oracle XML DB a été conçu pour stocker et extraire des objets XML via un chemin d'accès hiérarchique. Oracle XML DB optimise l'accès aux objets XML et les mises à jour de ces objets. Il permet également la gestion de la configuration et du contrôle d'accès de ces objets.
Configurer Oracle XML DB en vue de son utilisation dans votre environnement.
Créer une ressource dans Oracle XML DB. Pour modifier les privilèges de contrôle d'accès de cette ressource, sélectionnez-la dans l'arborescence du Navigateur, cliquez avec le bouton droit de la souris, puis choisissez l'option de menu Octroyer des privilèges à l'utilisateur/au rôle.
