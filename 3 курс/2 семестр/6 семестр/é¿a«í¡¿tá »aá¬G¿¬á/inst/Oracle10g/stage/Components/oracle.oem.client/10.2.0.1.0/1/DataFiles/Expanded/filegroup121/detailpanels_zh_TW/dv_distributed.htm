Distributed Database Management
Oracle supports data replication and messaging technologies to support distributed applications and distributed database systems.
Use In-Doubt Transactions folder to view failed two-phase commit transactions.
Use Database Links to create, edit and delete links between databases.
Use Streams folder to replicate data over a network of databases. You can monitor and administer your Streams environment. This folder is available for Oracle version 9.2 and above.
Use Advanced Queues folder to integrate applications by using messaging technology. You can monitor and administer your Advanced Queues environment.
Use Advanced Replication folder to replicate data amongst a group of databases (Multi Master Replication) or to replicate a snapshot of data between databases (Snapshot Replication). You can to monitor and administer your Advanced Replication environment.
