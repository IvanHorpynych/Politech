Ressourcen-Advisor
Instance Management>Ressourcen-Advisor
Oracle9i unterstützt Advisory-Verfahren, mit denen Sie Ihre Ressourcen im Hinblick auf effizienteste Benutzung optimieren können.
Mit dem Advisor für die Buffer-Cache-Größe bestimmen Sie die optimale Größe des Buffer-Cache.
Mit dem Advisor für die Shared Pool-Größe bestimmen Sie die optimale Größe für den Shared Pool, indem Sie dessen Benutzung durch den Library Cache verfolgen. Dieser Advisor ist ab Oracle 9.2 verfügbar.
Mit dem Program Global Area- (PGA) Advisor optimieren Sie den PGA-Speicher, der individuellen Server-Prozessen zugewiesen ist. Dieser Advisor ist ab Oracle 9.2 verfügbar.
Mit dem Mean-Time-To-Recover- (MTTR) Advisor bestimmen Sie den optimalen Wert für die maximale Zeit, die zur Wiederherstellung der Datenbank erforderlich ist. Dieser Advisor ist ab Oracle 9.2 verfügbar.
