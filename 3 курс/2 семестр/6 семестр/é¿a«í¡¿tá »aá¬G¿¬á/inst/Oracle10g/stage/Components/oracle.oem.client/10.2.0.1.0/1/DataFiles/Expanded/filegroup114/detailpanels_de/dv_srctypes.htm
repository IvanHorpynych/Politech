Quelltypen
Oracle unterstützt Objekte, bei denen es sich um Codeeinheiten handelt, die in SQL, PL/SQL, Java oder C geschrieben sind, um Anwendungslogik in der Datenbank zu entwickeln.
Verwenden Sie Package, um eine Gruppe mit zusammengehörigen Prozeduren und Funktionen zu erstellen oder zu durchsuchen, die als eine Einheit gespeichert ist.
Verwenden Sie Package Body, um alle öffentlichen und privaten Konstrukte des Packages zu erstellen oder zu durchsuchen. Sie können den Package Body kompilieren und debuggen.
Verwenden Sie Prozedur, um Code mit SQL und PL/SQL zu entwickeln.
Verwenden Sie Funktion, um Code zu entwickeln, der immer einen einzelnen Wert an den aufrufenden Benutzer zurückgibt.
Verwenden Sie Trigger, um Code zu entwickeln, der immer dann ausgeführt wird, wenn eine Tabelle oder View geändert wird oder wenn Benutzer- oder Datenbanksystemaktionen auftreten.
Verwenden Sie Java Source, um Java-Code in eine standardmäßige, plattform-unabhängige Gruppe von Byte-Codes zu kompilieren, die mit einer Java Virtual Machine (JVM) kommunizieren.
Laden Sie Java Class in die Datenbank zur Ausführung mit Java Virtual Machine (JVM) in der Datenbank.
