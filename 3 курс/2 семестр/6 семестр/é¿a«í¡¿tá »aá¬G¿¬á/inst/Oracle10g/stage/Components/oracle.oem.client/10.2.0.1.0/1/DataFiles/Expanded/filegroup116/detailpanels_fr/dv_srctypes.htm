Types de source
Oracle prend en charge des objets qui sont des unités de code écrits en langage SQL, PL/SQL, Java ou C pour développer une logique applicative dans la base de données.
Utilisez Package pour créer ou parcourir un groupe de procédures et de fonctions associées stocké en tant qu'unité.
Utilisez Corps de package pour créer ou parcourir toutes les structures publiques et privées du package. Vous pouvez compiler et déboguer le corps du package.
Utilisez Procédure pour développer du code en utilisant SQL et PL/SQL.
Utilisez Fonction pour développer un code renvoyant systématiquement une seule valeur à l'appelant.
Utilisez Déclencheur pour développer un code exécuté à chaque modification d'une table ou d'une vue, ou lorsque l'utilisateur ou le système de bases de données effectue certaines actions.
Utilisez Source Java pour compiler un code Java sous la forme d'un ensemble d'octets standard indépendants de toute plate-forme, interagissant avec une JVM (Java Virtual Machine).
Chargez la classe Java dans la base de données en vue de son exécution via la JVM (Java Virtual Machine) de la base de données.
CONSEIL Oracle recommande d'utiliser la console Enterprise Manager 10g pour l'administration et les diagnostics.
