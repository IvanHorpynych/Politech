Modèles de déploiement
Réplication des vues matérialisées>Modèles de déploiement
Les modèles de déploiement permettent de créer un ensemble de définitions de vue matérialisée sur un site maître et d'utiliser les paramètres de ces définitions pour que les vues matérialisées puissent être personnalisées pour chaque utilisateur.
Pour créer un modèle de déploiement sur un site maître :
  1. Cliquez sur Configurer les sites maîtres pour configurer le site maître, si nécessaire.
  2. Si vous souhaitez inclure dans le modèle des vues matérialisées actualisables, leurs tables de base doivent se trouver dans le groupe maître.
    Cliquez sur Créer un groupe maître pour créer le groupe maître qui gérera les objets répliqués.
  3. Cliquez sur Créer un modèle pour lancer l'assistant qui vous aidera à créer le modèle de déploiement.
Pour personnaliser le modèle de chaque utilisateur et le déployer sur les sites de vues matérialisées :
  1. Cliquez sur Génération de script de modèle pour terminer la génération des scripts de modèle hors ligne.
  2. Distribuez les fichiers d'instanciation à chaque utilisateur. Les utilisateurs peuvent exécuter le script sur le site de vues matérialisées à l'aide de SQL* Plus pour créer l'environnement de vues matérialisées.
Pour plus d'informations sur les modèles de déploiement, cliquez sur le bouton Aide.
