Copia de Seguridad y Recuperación
Gestión de Instancias>Copia de Seguridad y Recuperación
Las operaciones de copia de seguridad, de restauración y de recuperación de archivos de base de datos, como archivos de datos, archive logs, archivos de control y archivos de parámetros del servidor, se pueden realizar con la utilidad Oracle Recovery Manager (RMAN). La consola proporciona una sencilla interfaz a la utilidad RMAN para realizar estas tareas.
Utilice el Asistente de Copia de Seguridad para realizar una copia de seguridad de los objetos de base de datos.
Utilice el Asistente de Recuperación para recuperar objetos de base de datos.
Utilice el Asistente de Mantenimiento para configurar políticas de copia de seguridad y de retención en una base de datos destino o para registrar, restablecer o resincronizar bases de datos destino con el catálogo de recuperación.
Ejecute el trabajo Archivo de Comandos de RMAN mediante sus propios archivos de comandos de RMAN para ejecutar cualquier comando de RMAN y realizar operaciones de RMAN incluidas las operaciones de copia de seguridad y de recuperación.
