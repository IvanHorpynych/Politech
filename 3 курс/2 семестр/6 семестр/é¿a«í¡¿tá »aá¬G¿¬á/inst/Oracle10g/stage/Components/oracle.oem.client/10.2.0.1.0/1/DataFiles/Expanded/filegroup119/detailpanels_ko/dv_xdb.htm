XML 데이터베이스 관리
Oracle은 XML 데이터 모델을 기반으로 하는 고성능 객체 시스템인 XML 데이터베이스를 지원합니다. XML 데이터베이스는 계층 경로명을 통해 XML 객체를 저장하고 검색하도록 설계되었습니다. XML 데이터베이스는 액세스 권한을 최적화하며 여기에 저장된 XML 객체로 갱신합니다. 또한 이러한 객체에 대해 액세스 제어 및 구성 관리 기능도 제공합니다.
사용자 환경에 맞게 XML 데이터베이스를 구성합니다.
XML 데이터베이스에서 리소스를 생성합니다. 네비게이션 트리에서 이 리소스를 선택하고 마우스 오른쪽 단추를 눌러 사용자/롤에 권한 부여 메뉴 항목을 선택하여 이 리소스에 대한 액세스 제어 권한을 수정할 수 있습니다.
