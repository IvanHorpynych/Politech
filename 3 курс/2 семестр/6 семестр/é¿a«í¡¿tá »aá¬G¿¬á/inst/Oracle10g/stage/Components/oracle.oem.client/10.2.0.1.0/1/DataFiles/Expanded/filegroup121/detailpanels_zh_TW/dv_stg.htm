Storage Management
 
Use Storage Management to:
Create Tablespaces, Datafiles, Rollback Segments, and Redo Log Groups storage objects.
Add datafiles and rollback segments to a tablespace.
Drop storage objects.
Place objects online or offline.
Show dependencies on objects.
Perform backup, recovery and maintenance operations on the database files.
View Storage Layout. This is available for Oracle version 9.2 and above.
TIP Oracle recommends using the Enterprise Manager 10i Console for administration and diagnostics.
