Resource Advisors
Instance Management>Resource Advisors
Oracle9i supports advisory mechanisms to help you tune your resources for optimal use.
Use Buffer Cache Size Advisor to determine optmal size of the buffer cache.
Use Shared Pool Size Advisor to determine optimal Shared Pool size by tracking its use by the Library Cache. This advisor is available starting with Oracle 9.2.
Use Program Global Area (PGA) Advisor to tune PGA memory allocated to individual server processes. This advisor is available starting with Oracle 9.2.
Use Mean-Time-To-Recover (MTTR) Advisor to determine the optimal value for the maximum time needed to recover the database. This advisor is available starting with Oracle 9.2.
