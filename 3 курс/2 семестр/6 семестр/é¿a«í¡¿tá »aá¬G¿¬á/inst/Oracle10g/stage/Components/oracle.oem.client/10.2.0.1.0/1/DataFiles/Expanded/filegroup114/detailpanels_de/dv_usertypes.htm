Benutzertypen
Mit Oracle als einem objekt-relationen Datenbank-Management-System können Sie zusätzliche Datentypen definieren und innerhalb des relationalen Modells benutzen.
Verwenden Sie Objekttyp, um einen komplexen Typ zu erstellen, bei dem es sich um eine Abstraktion von wirklichen Entitäten handelt, mit denen Ihr Anwendungsprogramm arbeitet.
Verwenden Sie Array-Typ, um eine sortierte Gruppe von Datenelementen desselben Typs zu erstellen.
Verwenden Sie Tabellentyp, um eine unsortierte Gruppe von Datenelementen desselben Datentyps zu erstellen.
Verwenden Sie XML-Schema, um komplexe Typen und Speicherungselemente für XML-Dokumente basierend auf dem XML-Schema zu erstellen. Dieser Objekttyp ist für Oracle Version 9.2 und höher verfügbar.
TIPP Oracle empfiehlt die Benutzung der Enterprise Manager 10g Konsole für Administration und Diagnose.
