Replicación de Vista Materializada
 
Replicación de Vista Materializada permite crear réplicas completas o parciales de tablas en la base de datos origen (denominada Ubicación Maestra) en una o en varias bases de datos destino (denominadas Ubicaciones de Vista Materializada).
La replicación de vista materializada se puede llevar a cabo de formas diferentes. Haga clic en un enlace para obtener más información:
  • Utilice Plantillas de Despliegue para desplegar y administrar un entorno de vistas materializadas ampliamente distribuido de forma eficaz.
  • Utilice Vistas Materializadas para desplegar y administrar un entorno de vistas materializadas de sólo lectura.
  • Utilice Grupos de Vistas Materializadas para desplegar y administrar un entorno de vistas materializadas actualizable.
  • Utilice Grupos de Refrescamiento para refrescar varias vistas materializadas en un mismo paso.
Para obtener más información sobre la replicación de vistas materializadas, haga clic en el botón Ayuda.
