Gestion d'une base de données data warehouse
 
Une base de données data warehouse est une base de données relationnelle destinée à l'interrogation et à l'analyse, plutôt qu'au traitement des transactions. Oracle prend en charge l'analyse complexe des données de base de données data warehouse via des applications OLAP (on-line analytical processing). Oracle fournit également un mécanisme permettant d'améliorer les performances d'une base de données data warehouse à l'aide de récapitulatifs.
 
Dans le Navigateur, sélectionnez le dossier Gestion des récapitulatifs pour la gestion des récapitulatifs.
