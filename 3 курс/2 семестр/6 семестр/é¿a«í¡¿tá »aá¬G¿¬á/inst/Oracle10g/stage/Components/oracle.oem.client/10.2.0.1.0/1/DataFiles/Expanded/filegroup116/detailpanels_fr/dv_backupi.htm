Sauvegarde et récupération
Instance Management>Sauvegarde et récupération
Les opérations de sauvegarde, de restauration et de récupération sur des fichiers de base de données (fichiers de données, journaux archivés, fichier de contrôle ou fichier de paramètres côté serveur) peuvent être exécutées avec l'utilitaire Oracle Recovery Manager (RMAN). La console fournit une interface vers RMAN, facile à utiliser, pour effectuer ces tâches.
Utilisez l'assistant Sauvegarde pour sauvegarder des objets de base de données.
Utilisez l'assistant Récupération pour récupérer des objets de base de données.
Utilisez l'assistant Maintenance pour définir des règles de sauvegarde et de conservation dans une base de données cible, ou pour inscrire, réinitialiser ou resynchroniser des bases de données cible avec le catalogue de récupération.
Soumettez un travail de script RMAN à l'aide de vos propres scripts RMAN pour exécuter une commande RMAN et effectuer des opérations RMAN, comme la sauvegarde et la récupération.
