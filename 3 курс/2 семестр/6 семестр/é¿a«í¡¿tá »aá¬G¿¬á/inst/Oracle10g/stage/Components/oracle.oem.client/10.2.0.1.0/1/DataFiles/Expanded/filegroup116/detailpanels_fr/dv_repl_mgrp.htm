Groupes de vues matérialisées
Réplication des vues matérialisées>Groupes de vues matérialisées
Un groupe de vues matérialisées contient la totalité ou une partie des objets du groupe maître correspondant. Les vues matérialisées d'un groupe de vues matérialisées peuvent être actualisables ou en lecture seule.
Pour créer un groupe de vues matérialisées :
  1. Cliquez sur Configurer les sites maîtres pour configurer le site maître, si nécessaire.

  2. Cliquez sur Créer un groupe maître pour créer un groupe maître (si nécessaire) sur le site maître.

  3. Cliquez sur Créer un groupe de vues matérialisées pour lancer l'assistant qui créera le groupe de vues matérialisées sur le site de vues matérialisées.
Pour plus d'informations sur les groupes de vues matérialisées, cliquez sur le bouton Aide.
