Gestores Simultáneos
 
Los gestores simultáneos planifican solicitudes de procesamiento en bases de datos de Aplicaciones Oracle. La Consola permite:
Crear eventos para controlar los gestores simultáneos en caso de que necesiten intervención.
Planificar trabajos por lotes para corregir esas condiciones.
Planificar períodos de interrupción de notificaciones o interrumpir todas las notificaciones de ese gestor simultáneo.
Utilizar Oracle Capacity Planner para analizar tendencias históricas y la capacidad futura de proyectos.
Utilizar los gráficos de Oracle Performance Manager para analizar estadísticas del gestor simultáneo.
Generar informes de datos históricos a partir de gestores simultáneos, programas simultáneos y del procesamiento de programas simultáneos.
Suprimir un gestor simultáneo del árbol del navegador.
