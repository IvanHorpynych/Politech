Resource Advisor
Gestione istanze>Resource Advisor
Oracle9i include meccanismi di avviso che contribuiscono alla regolazione delle risorse disponibili per il loro utilizzo ottimale.
Utilizzare Buffer Cache Size Advisor per determinare le dimensioni ottimali della buffer cache.
Utilizzare Shared Pool Size Advisor per determinare le dimensioni ottimali dello shared pool monitorandone l'utilizzo da parte della Library Cache. Questo avviso è disponibile a partire da Oracle 9.2.
Utilizzare Program Global Area (PGA) Advisor per regolare la memoria PGA allocata a singoli processi del server. Questo avviso è disponibile a partire da Oracle 9.2.
Utilizzare Mean-Time-To-Recover (MTTR) Advisor per determinare il valore ottimale per il tempo massimo necessario al recupero del database. Questo avviso è disponibile a partire da Oracle 9.2.
