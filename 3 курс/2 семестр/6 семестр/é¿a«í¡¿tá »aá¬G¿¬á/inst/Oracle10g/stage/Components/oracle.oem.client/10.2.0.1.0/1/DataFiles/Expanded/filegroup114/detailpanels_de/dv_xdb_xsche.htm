XML-Schema-basierte Objekte
Verwaltung von XML Database>XML-Schema-basierte Objekte
XML Schema ist eine Sprache zur Schemadefinition (auch in XML), mit der die Struktur und Semantik von entsprechenden XML-Instance-Dokumenten beschrieben werden kann.
Mit Oracle 9.2 können Sie
Ein XML-Schema mit der Datenbank registrieren entweder mit einer bestehenden Ressource von XML Database oder einer Schemadatei von XML Database, auf die lokal oder über das Internet zugegriffen werden kann.
Eine strukturierte Speicherung basierend auf dem XML-Schema erstellen.
Hierzu gibt es zwei Möglichkeiten:
Erstellen Sie eine Tabelle basierend auf der Struktur, die in dem XML-Schema definiert ist.
Erstellen Sie eine Tabelle mit einer Spalte, die auf der im XML-Schema definierten Struktur basiert.
Das Erstellen einer XML-Schema-basierten Speicherung bietet folgende Vorteile:
  • XML-Instance-Dokumente, die in dieser Speicherung abgelegt sind, belegen wesentlich weniger Platz
  • Zugriff zu bestimmten Datenteilen über XPath ist hoch-optimiert
  • Update-Vorgänge von Teilen des XML-Instance-Dokuments sind wesentlich verbessert
Bestehende Daten durch Erstellen von Views basierend auf einem XML-Schema in XML-Formate umsetzen.
Sie können auch funktionale Indizes auf XML-Schema-basierten Tabellen mit XPath-Ausdrücken erstellen.
