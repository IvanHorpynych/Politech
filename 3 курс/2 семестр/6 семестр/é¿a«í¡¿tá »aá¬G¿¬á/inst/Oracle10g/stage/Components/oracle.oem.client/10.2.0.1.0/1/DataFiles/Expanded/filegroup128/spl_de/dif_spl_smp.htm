Mit DB Diff können Sie Objektdefinitionen vergleichen. Sie können Definitionen aus zwei Datenbanken, zwei Baselines, einer Datenbank und einer Baseline oder zwei Schemata innerhalb einer einzelnen Datenbank/Baseline vergleichen.

Danach können Sie mit dem Synchronisations-Assistenten von DB Diff Datenbank- oder Schemaobjektdefinitionen ändern, damit diese den jeweiligen Objektdefinitionen entsprechen.
