Objets basés sur un schéma XML
Gestion d'Oracle XML DB>Objets basés sur un schéma XML
Un schéma XML est un langage de définition de schéma (également dans XML) servant à décrire la structure et la sémantique de documents d'instance XML conformes.
Oracle 9.2 vous permet d'effectuer les opérations suivantes :
Inscription d'un schéma XML dans la base de données à l'aide d'une ressource Oracle XML DB existante, ou via un fichier de schéma Oracle XML DB accessible localement ou via Internet.
Création d'un stockage structuré basé sur un schéma XML.
Vous avez le choix entre les deux méthodes suivantes :
Création d'une table basée sur la structure définie dans le schéma XML
Création d'une table à colonnes basée sur la structure définie dans le schéma XML
La création d'un stockage basé sur un schéma XML présente les avantages suivants :
  • Les documents d'instance XML présents dans ce stockage occupent considérablement moins d'espace.
  • L'accès à des éléments de données spécifiques, exprimés via XPath, est considérablement optimisé.
  • Les opérations de mise à jour effectuées sur certaines parties d'un document d'instance XML sont largement améliorées.
Répartir les données existantes au format XML en créant des vues basées sur le schéma XML
Vous pouvez également créer des index fonctionnels sur les tables basées sur le schéma XML à l'aide d'expressions XPath.
