Schema Management
 
Verwenden Sie Schema Management für folgende Zwecke:
Erstellen, ändern und löschen Sie Schemaobjekte.
Zeigen Sie Abhängigkeiten von Schemaobjekten an.
Mit SQL Scratchpad führen Sie die SQL-Anweisungen aus.
Mit dem Export-Assistenten exportieren Sie die Daten in eine Exportdatei im Binärformat. Mit dem Import-Assistenten importieren Sie Daten aus dieser Datei.
Mit dem Analyse-Assistenten analysieren Sie Datenbankobjekte.
Bearbeiten Sie mehrere Objekte, indem Sie sie aus der Liste auswählen.
Suchen Sie Datenbankobjekte.
Generieren Sie das DDL-Skript für ein Objekt.
TIPP Oracle empfiehlt die Benutzung der Enterprise Manager 10g Konsole für Administration und Diagnose.
