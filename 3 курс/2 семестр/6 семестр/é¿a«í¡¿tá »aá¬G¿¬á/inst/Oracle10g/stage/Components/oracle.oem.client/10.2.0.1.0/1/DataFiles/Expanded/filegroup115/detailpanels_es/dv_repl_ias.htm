Plantillas de iAS
Replicación de Vista Materializada>Plantillas de iAS
Una Plantilla de iAS contiene un juego de objetos que se van a instanciar en una base de datos de capa media. Las aplicaciones de iAS pueden almacenar objetos a los que se accede frecuentemente en una base de datos de capa media para mejorar el rendimiento y la escalabilidad de la aplicación.
Para crear una plantilla de iAS en una base de datos backend:
  1. Cree un usuario propietario de plantilla que tenga privilegios CONNECT y SELECT_CATALOG_ROLE (o privilegios de DBA). A continuación, conéctese como el propietario de la plantilla.
  2. Cree una plantilla de iAS que defina los objetos que se van a crear en las ubicaciones de capa media.
  3. Prepare la plantilla de exportación para configurar la plantilla de forma que se pueda instanciar.
  4. Exporte la plantilla para guardar la definición de plantilla en un archivo de volcado de exportación.
  5. Copie el archivo de volcado de exportación en las máquinas de capa media en las que desee instanciar la plantilla.
Para desplegar la plantilla en ubicaciones de capa media:
  1. Conéctese a la base de datos de capa media y cree un usuario propietario de plantilla que tenga privilegios de DBA.
  2. Vuelva a conectarse a la base de datos de capa media como el propietario de la plantilla.
  3. Importe la plantilla para instanciarla.
  4. Defina la política de refrescamiento del grupo de refrescamiento asociado a la plantilla que se acaba de instanciar.
Una vez que ha instanciado la plantilla en todas las bases de datos de capa media, conéctese a la base de datos backend y realice la limpieza post instanciación.

Para obtener más información sobre las plantillas de iAS, haga clic en el botón Ayuda.
