Backup and Recovery
Storage Management>Backup and Recovery
Backup, restore and recovery operations on database files such as datafiles, archived logs, control file and server parameter file can be performed with Oracle Recovery Manager (RMAN) utility. Console provides an easy-to-use interface to the RMAN utility to perform these tasks. See the Backup Management Quick Tour to learn more about Console support for backup and recovery.
Use Backup Wizard to backup database objects.
Use Recovery Wizard to recover database objects.
Use Maintenance Wizard to set up backup and retention policies in a target database, or register, reset, or re-synchronize target databases with the recovery catalog.
Submit RMAN Script job by using your own RMAN scripts to execute any RMAN command and perform RMAN operations including Backup and Recovery operations.
