Plan Editor
Change Manager>Plan Editor
Plan Editor è un componente di Change Manager che viene avviato quando si crea o modifica un piano di modifica. La modalità di lavoro consiste nel procedere dall'alto verso il basso nella vista della struttura di Plan Editor per definire ciascun elemento del piano di modifica:
  1. Creare le richieste di modifica.
  2. Specificare i database di destinazione, ossia i database in cui si desiderano apportare le modifiche.
  3. Generare lo script.
  4. Eseguire lo script.
