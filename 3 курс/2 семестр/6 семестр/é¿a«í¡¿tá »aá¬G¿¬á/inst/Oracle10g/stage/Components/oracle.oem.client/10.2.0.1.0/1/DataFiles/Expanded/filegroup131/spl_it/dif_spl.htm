Confronta oggetti del database
Change Manager>Confronta oggetti del database
Il Confronto guidato oggetti del database consente di confrontare le definizioni degli oggetti. È possibile confrontare le definizioni di due database, di due baseline, di un database e di una baseline oppure di due schemi dello stesso database o della stessa baseline.

Successivamente è possibile utilizzare la Sincronizzazione guidata per modificare le definizioni degli oggetti del database o dello schema in modo che le definizioni degli oggetti di tali elementi corrispondano.
