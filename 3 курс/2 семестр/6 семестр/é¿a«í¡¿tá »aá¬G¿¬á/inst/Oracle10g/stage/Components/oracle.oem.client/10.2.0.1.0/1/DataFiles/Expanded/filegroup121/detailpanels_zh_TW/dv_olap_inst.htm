OLAP Services Configuration
OLAP Management > OLAP Services Configuration
OLAP services are database processes that process analytical information while allowing load-sharing. They can be managed and configured using OLAP Services Instance Manager. Use this tool to:
 
Start, stop, and pause OLAP services.
Create new services and remove existing services.
Change configuration parameters.
Monitor client sessions.
View system logs.
