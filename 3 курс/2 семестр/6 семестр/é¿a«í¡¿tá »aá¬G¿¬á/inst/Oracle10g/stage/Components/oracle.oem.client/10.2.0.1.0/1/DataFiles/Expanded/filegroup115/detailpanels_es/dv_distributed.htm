Gestión de Bases de Datos Distribuidas
Oracle soporta las tecnologías de replicación de datos y de mensajería para dar soporte a aplicaciones distribuidas y a sistemas de bases de datos distribuidas.
Utilice la carpeta Transacciones Dudosas para ver transacciones de validación en dos fases con fallos.
Utilice Enlaces de Base de Datos para crear, editar y suprimir enlaces entre bases de datos.
Utilice la carpeta Streams para replicar datos a través de una red de bases de datos. Puede controlar y administrar su entorno de Streams. Esta carpeta está disponible en Oracle versión 9.2 y superiores.
Utilice la carpeta Colas Avanzadas para integrar aplicaciones mediante el uso de tecnologías de mensajería. Puede controlar y administrar su entorno de Colas Avanzadas.
Utilice la carpeta Replicación Avanzada para replicar datos entre un grupo de bases de datos (Replicación de Varios Maestros) o para replicar una instantánea de datos entre bases de datos (Replicación de Instantánea). Puede controlar y administrar su entorno de Replicación Avanzada.
