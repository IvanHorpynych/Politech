iAS-Vorlagen
Materialized View-Replikation>iAS-Vorlagen
Eine iAS-Vorlage enthält eine Reihe von Objekten, die auf einer Middle-Tier-Datenbank instanziiert werden. iAS-Anwendungen können Objekte, auf die häufig zugegriffen wird, in einer Middle-Tier-Datenbank speichern, um die Anwendungsleistung und -skalierbarkeit zu verbessern.
So erstellen Sie eine iAS-Vorlage auf einer Back-End-Datenbank:
  1. Erstellen Sie einen Vorlageneigentümer, der CONNECT-Berechtigungen und die SELECT_CATALOG_ROLE (oder DBA-Berechtigungen) hat. Danach melden Sie sich als Vorlageneigentümer an.
  2. Erstellen Sie eine iAS-Vorlage, mit der die Objekte definiert werden, die auf den Rechnern der mittleren Schicht erstellt werden sollen.
  3. Bereiten Sie die Vorlage für den Export vor, um die Vorlage so einzurichten, dass sie instanziiert werden kann.
  4. Exportieren Sie die Vorlage, um die Vorlagendefinition in einer Export-Dump-Datei zu speichern.
  5. Kopieren Sie die Export-Dump-Datei in die Rechner der mittleren Schicht, auf denen die Vorlage instanziiert werden soll.
So wenden Sie die Vorlage auf Rechnern der mittleren Schicht an:
  1. Melden Sie sich bei der Middle-Tier-Datenbank an, und erstellen Sie einen Vorlageneigentümer, der DBA-Berechtigungen hat.
  2. Melden Sie sich erneut als Vorlageneigentümer bei der Middle-Tier-Datenbank an.
  3. Importieren Sie die Vorlage, um die Vorlage zu instanziieren.
  4. Legen Sie die Aktualisieren-Richtlinien für die Aktualisierengruppe fest, die mit der neu instanziierten Vorlage verknüpft ist.
Nachdem Sie die Vorlage auf allen Middle-Tier-Datenbanken instanziiert haben, melden Sie sich bei der Back-End-Datenbank an und führen den Aufräumenvorgang nach der Instanziierung durch.
