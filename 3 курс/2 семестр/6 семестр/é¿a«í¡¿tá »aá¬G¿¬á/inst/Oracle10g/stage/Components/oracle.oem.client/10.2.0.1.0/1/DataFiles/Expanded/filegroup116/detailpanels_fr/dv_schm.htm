Schema Management
 
Utilisez Schema Management pour les opérations suivantes :
Création, modification et suppression d'objets de schéma.
Affichage des dépendances portant sur les objets de schéma.
Utilisation de SQL Scratchpad pour exécuter des instructions SQL.
Utilisation de l'assistant Export pour exporter les données vers un fichier d'export au format binaire et utilisation de l'assistant Import pour importer des données à partir de ce fichier.
Utilisation de l'assistant Analyse pour analyser des objets de base de données.
Exécution d'opérations sur plusieurs objets sélectionnés dans la liste.
Recherche d'objets de base de données.
Génération du script DDL pour un objet.
CONSEIL Oracle recommande d'utiliser la console Enterprise Manager 10g pour l'administration et les diagnostics.
