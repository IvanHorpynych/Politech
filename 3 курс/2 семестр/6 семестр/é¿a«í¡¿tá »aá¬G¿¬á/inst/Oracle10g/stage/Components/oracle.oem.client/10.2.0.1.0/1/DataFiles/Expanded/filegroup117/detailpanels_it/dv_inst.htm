Gestione istanze
 
Utilizzare Gestione istanze per effettuare le operazioni riportate di seguito.
Visualizzare e modificare i valori dei parametri di istanza.
Ottimizzare le risorse del database utilizzando i Resource Advisor.
Gestire le sessioni degli utenti e visualizzare il codice SQL attualmente in esecuzione con il relativo explain plan.
Monitorare i lock e le sessioni che utilizzano le maggiori quantità di risorse e lo stato del database (se è installato Diagnostics Pack).
Controllare le operazioni con tempi di esecuzione lunghi.
Controllare le risorse di elaborazione tramite piani risorse.
SUGGERIMENTO Per le operazioni di amministrazione e diagnostica, la Oracle consiglia di utilizzare Enterprise Manager 10g Console.
