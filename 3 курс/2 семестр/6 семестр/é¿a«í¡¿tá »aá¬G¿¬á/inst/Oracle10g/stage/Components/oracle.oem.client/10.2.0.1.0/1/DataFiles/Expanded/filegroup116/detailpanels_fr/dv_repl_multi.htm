Réplication à plusieurs maîtres
 
La réplication à plusieurs maîtres permet à plusieurs sites (ou sites maîtres), agissant en tant qu'homologues, de gérer des groupes d'objets de base de données répliqués.
Pour configurer la réplication à plusieurs maîtres :
  1. Cliquez sur Configurer les sites maîtres pour lancer un assistant qui vous aidera à configurer un environnement de réplication à plusieurs maîtres.

  2. Cliquez sur Créer un groupe maître pour créer un groupe maître qui gérera les objets répliqués. Le nouveau groupe apparaîtra dans le Navigateur et une demande sera soumise pour lancer le processus de réplication.

  3. Dans le Navigateur, sélectionnez le groupe maître que vous venez de créer. La réplication multimaître fonctionne si le statut du groupe maître est Démarré.
Pour plus d'informations sur la réplication à plusieurs maîtres, cliquez sur le bouton Aide.
