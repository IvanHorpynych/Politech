Bases de données
 
Le dossier Base de données contient toutes les bases de données auxquelles vous pouvez accéder. Vous pouvez développer une base de données particulière pour administrer les éléments suivants :
Des instances, notamment leur initialisation.
Des schémas, notamment les tables, les index et tout autre objet schéma.
La sécurité, notamment les comptes utilisateur, les rôles et les privilèges.
Le stockage, notamment les espaces disque logiques, les fichiers de données et les segments d'annulation.
La distribution, notamment la gestion des liens de base de données, des flux de données, des files d'attente avancées et de la réplication avancée.
Base de données Data Warehouse, notamment la gestion des récapitulatifs et la gestion d'OLAP.
Les espaces de travail, environnement virtuel que des utilisateurs peuvent partager pour apporter aux données des modifications stockées dans des fichiers dotés d'attributs de gestion des versions.
D'autres fonctions de base de données, selon les produits installés avec votre base de données.
Présentation de la console Oracle Enterprise Manager 10g
Oracle recommande d'utiliser la console Enterprise Manager 10g pour les fonctions suivantes :
Des instances, notamment leur démarrage, leur arrêt et leur initialisation.
Des schémas, notamment les tables, les index et tout autre objet schéma.
La sécurité, notamment les comptes utilisateur, les rôles et les privilèges.
Le stockage, notamment les espaces disque logiques, les fichiers de données et les segments d'annulation.
Utilisez la console Enterprise Manager 10g pour les fonctions suivantes :
Des instances, notamment leur démarrage, leur arrêt et leur initialisation.
Des outils de base de données, notamment la gestion des sauvegardes, la gestion des données et l'analyse d'objets.
Des fonctionnalités de diagnostic, notamment l'aperçu des performances pour les verrous de base de données, et les sessions et les instructions SQL les plus consommatrices de ressources.
