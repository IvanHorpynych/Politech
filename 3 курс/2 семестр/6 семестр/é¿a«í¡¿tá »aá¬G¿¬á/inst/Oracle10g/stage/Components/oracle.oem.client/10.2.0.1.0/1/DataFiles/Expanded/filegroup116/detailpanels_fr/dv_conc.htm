Gestionnaires de traitements simultanés (GTS)
 
Les gestionnaires de traitements simultanés programment les demandes de traitement sur les bases de données Oracle Applications. La console permet d'effectuer les tâches suivantes :
Création d'événements pour surveiller les conditions nécessitant une intervention sur les gestionnaires de traitements simultanés (GTS)
Programmation de travaux par lots pour corriger ces conditions
Programmation de périodes de coupure de notification ou coupure de toutes les notifications de ce gestionnaire de traitements simultanés
Utilisation de Capacity Planner pour analyser les tendances historiques et la capacité future du projet
Utilisation de graphiques Performance Manager pour analyser les statistiques du gestionnaire de traitements simultanés (GTS)
Génération de rapports des données historiques des gestionnaires de traitements simultanés, des programmes simultanés et du traitement de ces programmes
Suppression d'un gestionnaire de traitements simultanés (GTS) dans l'arborescence du Navigateur
