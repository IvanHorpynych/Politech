Backup e recupero
Gestione memoria>Backup e recupero
Le operazioni di backup, ripristino e recupero sui file del database come i file di dati, i log archiviati, il control file e il file dei parametri del server possono essere eseguiti con la utility di Oracle Recovery Manager (RMAN). Console fornisce alla utility RMAN un'interfaccia di facile utilizzo per eseguire questi task.
Utilizzare Backup guidato per eseguire il backup degli oggetti del database.
Utilizzare Recupero guidato per recuperare gli oggetti del database.
Utilizzare Gestione guidata per impostare i criteri di backup e di conservazione in un database di destinazione oppure per registrare, reimpostare o risincronizzare il database di destinazione con il recovery catalog.
Sottomettere un job di script RMAN utilizzando script RMAN personalizzati per eseguire comandi e operazioni RMAN, compresi il backup e il recupero.
