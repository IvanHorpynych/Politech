Gestione dei warehouse
 
Per data warehouse si intende un database relazionale progettato per l'esecuzione di query e l'analisi piuttosto che per l'elaborazione delle transazioni. Oracle supporta l'analisi complessa dei dati del warehouse mediante le applicazioni di elaborazione analitica OLAP (On-Line Analytical Processing). Oracle fornisce inoltre un meccanismo per il miglioramento delle prestazioni del warehouse costituito dall'utilizzo dei riepiloghi.
 
Per amministrare i riepiloghi, selezionare la cartella Gestione riepiloghi nel Navigator.
