Gestion d'une base de données distribuée
Oracle prend en charge les technologies de réplication de données et de messagerie pour la prise en charge des applications distribuées et des systèmes de bases de données distribuées.
Utilisez le dossier Transactions en suspens pour visualiser les transactions de validation en deux phases ayant échoué.
Utilisez Liens de base de données pour créer, modifier et supprimer des liens entre des bases de données.
Utilisez le dossier Flux de données pour répliquer des données sur un réseau de bases de données. Vous pouvez surveiller et gérer votre environnement Oracle Streams. Ce dossier est disponible pour Oracle versions 9.2 et supérieures.
Utilisez le dossier Files d'attente avancées pour intégrer des applications en utilisant la technologie de messagerie. Vous pouvez surveiller et gérer votre environnement de files d'attente avancées.
Utilisez le dossier Réplication avancée pour répliquer des données au sein d'un groupe de bases de données (Réplication à plusieurs maîtres) ou pour répliquer un cliché de données entre des bases de données (Réplication des vues matérialisées). Vous pouvez surveiller et gérer votre environnement de réplication avancée.
