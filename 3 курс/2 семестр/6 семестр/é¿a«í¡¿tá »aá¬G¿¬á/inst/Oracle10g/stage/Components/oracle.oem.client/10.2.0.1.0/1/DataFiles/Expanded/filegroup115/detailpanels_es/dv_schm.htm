Gestión de Esquemas
 
Utilice Gestión de Esquemas para:
Crear, modificar y borrar objetos de esquema.
Mostrar dependencias de objetos de esquema.
Utilizar SQL Scratchpad para ejecutar sentencias SQL.
Utilizar el Asistente de Exportación para exportar los datos a un archivo de exportación de formato binario y el Asistente de Importación para importar datos desde este archivo.
Utilizar el Asistente de Análisis para analizar objetos de base de datos.
Trabajar en varios objetos seleccionándolos de la lista.
Buscar objetos de base de datos.
Generar el archivo de comandos DDL de un objeto.
CONSEJO Oracle recomienda utilizar la Consola de Enterprise Manager 10g para llevar a cabo tareas de administración y de diagnóstico.
