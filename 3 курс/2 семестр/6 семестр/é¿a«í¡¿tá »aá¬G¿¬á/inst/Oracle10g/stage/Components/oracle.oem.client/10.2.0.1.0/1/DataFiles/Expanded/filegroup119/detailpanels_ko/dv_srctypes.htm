소스 유형
Oracle은 응용 프로그램 논리를 데이터베이스로 개발하는 데 SQL, PL/SQL, Java 또는 C를 사용하여 작성한 코드 단위인 객체를 지원합니다.
패키지를 사용하여 한 단위로 저장된 관련 프로시저 및 함수의 한 그룹을 생성하거나 찾아볼 수 있습니다.
패키지 본문을 사용하여 패키지의 모든 공용 및 전용 구조를 생성하거나 찾아볼 수 있습니다. 패키지 본문을 컴파일하고 디버그할 수 있습니다.
프로시저를 통해 SQL 및 PL/SQL을 사용하여 코드를 개발할 수 있습니다.
함수를 사용하여 호출자에게 항상 단일 값을 반환하는 코드를 개발할 수 있습니다.
트리거를 사용하여 테이블이나 뷰가 수정되는 경우 또는 어떤 사용자 작업이나 데이터베이스 시스템 작업이 발생할 때 실행되는 코드를 개발할 수 있습니다.
Java 소스를 사용하여 Java 코드를 JVM(Java Virtual Machine)과 상호 작용하는 플랫폼 독립적인 일련의 표준 바이트 코드로 컴파일할 수 있습니다.
데이터베이스의 JVM(Java Virtual Machine)을 사용하여 실행을 위해 Java 클래스를 데이터베이스로 로드할 수 있습니다.
참고 Oracle은 관리 및 진단에 Enterprise Manager 10g Console을 사용할 것을 권장합니다.
