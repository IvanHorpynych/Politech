Asesores de Recursos
Gestión de Instancias>Asesores de Recursos
Oracle9i soporta mecanismos asesores que permiten ajustar los recursos para un uso óptimo.
Utilice el Asesor de Tamaño de Caché de Buffers para determinar el tamaño óptimo de la caché de buffers.
Utilice el Asesor de Tamaño de Conjunto Compartido para determinar el tamaño óptimo del conjunto compartido mediante el seguimiento de su uso por parte de la caché de biblioteca. Este asesor está disponible al iniciar con Oracle 9.2.
Utilice el Asesor de PGA (Área Global de Programa) para ajustar la memoria PGA asignada a cada uno de los procesos de servidor. Este asesor está disponible al iniciar con Oracle 9.2.
Utilice el Asesor de MTTR (Tiempo Medio para la Recuperación) para determinar el valor óptimo del tiempo máximo necesario para recuperar la base de datos. Este asesor está disponible al iniciar con Oracle 9.2.
