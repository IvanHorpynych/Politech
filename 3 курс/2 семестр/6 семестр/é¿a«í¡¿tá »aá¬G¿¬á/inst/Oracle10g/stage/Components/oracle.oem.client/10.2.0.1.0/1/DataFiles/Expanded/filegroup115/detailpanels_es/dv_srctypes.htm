Tipos de Origen
Oracle soporta objetos que sean unidades de código escritas en SQL, PL/SQL, Java o C para desarrollar la lógica de la aplicación en la base de datos.
Utilice Paquete para crear o examinar un grupo de procedimientos y funciones relacionados que se almacena como una unidad.
Utilice Cuerpo del Paquete para crear o examinar todas las creaciones públicas y privadas del paquete. El cuerpo del paquete se puede compilar y depurar.
Utilice Procedimiento para desarrollar código mediante SQL y PL/SQL.
Utilice Función para desarrollar el código que siempre devuelve un solo valor a la llamada.
Utilice Disparador para desarrollar el código que se ejecuta siempre que se modifica una tabla o una vista o cuando se producen algunas acciones de usuario o del sistema de base de datos.
Utilice Origen Java para compilar el código Java en un juego de códigos de byte estándar e independiente de la plataforma que interactúa con una máquina virtual de Java (JVM).
Cargue la clase Java en la base de datos para su ejecución mediante la máquina virtual de Java (JVM) en la base de datos.
CONSEJO Oracle recomienda utilizar la Consola de Enterprise Manager 10g para llevar a cabo tareas de administración y de diagnóstico.
