Source Types
Oracle supports objects which are units of code written in SQL, PL/SQL, Java or C to develop application logic into the database.
Use Package to create or browse a group of related procedures and functions that is stored as a unit.
Use Package Body to create or browse all public and private constructs of the package. You can compile and debug the package body.
Use Procedure to develop code using SQL and PL/SQL.
Use Function to develop code that always returns a single value to the caller.
Use Trigger to develop code that executes whenever a table or view is modified or when some user actions or database system actions occur.
Use Java Source to compile Java code to a standard, platform-independent set of bytecodes, which interacts with a Java virtual machine (JVM).
Load Java Class into the database for execution using Java virtual machine (JVM) in the database.
TIP Oracle recommends using the Enterprise Manager 10i Console for administration and diagnostics.
