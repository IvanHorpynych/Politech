La opción Diferencia de Base de Datos permite comparar definiciones de objetos. Puede comparar definiciones de dos bases de datos, dos líneas base, una base de datos y una línea base o dos esquemas en una sola base de datos/línea base.

A continuación, puede utilizar el Asistente de Sincronización de Diferencias de Base de Datos para modificar las definiciones de objetos de base de datos o esquema para que coincidan con sus correspondientes definiciones de objetos.
