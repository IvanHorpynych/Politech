XML Schemaに基づくオブジェクト
XMLデータベース管理 > XML Schemaに基づくオブジェクト
XML Schemaは、XMLインスタンス文書に準拠する構造およびセマンティクスを記述するために使用されるXMLのスキーマ定義言語です。
Oracle9i リリース9.2では、次のことができます。
既存のXMLデータベース・リソースを使用するか、ローカルまたはインターネットでアクセス可能なXMLデータベースのスキーマ・ファイルを使用して、データベースにXML Schemaを登録します。
XML Schemaに基づいて構造化記憶域を作成します。
これは、2つのうちのいずれかの方法で行います。
XML Schemaに定義された構造に基づいて表を作成します。
XML Schemaに定義された構造に基づいた列を含む表を作成します。
XML Schemaに基づいて作成された記憶域には、次の利点があります。
  • この記憶域に格納されるXMLインスタンス文書に必要な領域が、非常に少なくてすみます。
  • XPathで表現される特定のデータの一部へのアクセスが、大幅に最適化されます。
  • XMLインスタンス文書の一部への更新操作が、大幅に改善されます。
XML Schemaに基づいてビューを作成することによって、既存のデータをXML形式にラップします。
また、XPath表現を使用することによって、XML Schemaに基づく表にファンクション索引を作成できます。
