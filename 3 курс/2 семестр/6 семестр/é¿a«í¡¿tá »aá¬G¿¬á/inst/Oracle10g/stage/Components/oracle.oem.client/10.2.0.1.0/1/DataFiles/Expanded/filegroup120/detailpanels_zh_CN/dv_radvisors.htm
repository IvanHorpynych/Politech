Resource Advisor
实例管理 > Resource Advisor
Oracle9i 支持顾问机制, 帮助您调整资源以达到最优使用。
使用 Buffer Cache Size Advisor 可确定缓冲区高速缓存的最优大小。
使用 Shared Pool Size Advisor 可通过跟踪库高速缓存对共享池的使用来确定其最优大小。此 Advisor 自 Oracle 9.2 后可用。
使用 Program Global Area (PGA) Advisor 可调整分配给各个服务器进程的 PGA 内存。此 Advisor 自 Oracle 9.2 起可用。
使用 Mean-Time-To-Recover (MTTR) Advisor 可确定恢复数据库所需的最大时间的最优值。此 Advisor 自 Oracle 9.2 起可用。
