OLAP Metadata Objects
OLAP Management > OLAP Metadata Objects
To perform analytical processing, the data warehouse must be preconfigured with OLAP metadata objects, and access to the data warehouse must be optimized by creating materialized views. OLAP metadata objects are as follows:

  • Dimension Objects organize data warehouse dimension tables into levels, attributes, and hierarchies. To create these objects, launch the Dimension wizard

  • Cubes are logical representations of multi-dimensional data, and must include at least one dimension. To create cubes, launch the Cube wizard

  • Measure Folders organize measures into manageable groups. To create measure folders, launch the Measure Folder dialog
When a cube is selected in the Navigator tree, right-click it to launch the following tools:
  • Summary Advisor wizard - This wizard recommends materialized views that optimize performance of OLAP client applications that access data defined by a cube

  • Cube Viewer - This application displays the multi-dimensional data represented by the cube
