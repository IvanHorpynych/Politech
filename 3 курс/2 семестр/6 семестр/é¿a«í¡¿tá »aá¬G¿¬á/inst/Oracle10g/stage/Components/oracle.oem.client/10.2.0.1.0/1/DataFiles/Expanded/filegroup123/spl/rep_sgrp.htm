A Materialized View Group contains all or some of the objects in a corresponding master group. Materialized views in a materialized view group can be updatable or read-only.
To create a materialized view group:
  1. Click Setup Master Sites... to set up the master site, if not yet complete.

  2. Click Create Master Group... to create a master group on the master site, if the master group does not exist.

  3. Click Create Materialized View Group... to launch a wizard that creates the materialized view group on the materialized view site.
To learn more about Materialized View Groups, click the Help button.
