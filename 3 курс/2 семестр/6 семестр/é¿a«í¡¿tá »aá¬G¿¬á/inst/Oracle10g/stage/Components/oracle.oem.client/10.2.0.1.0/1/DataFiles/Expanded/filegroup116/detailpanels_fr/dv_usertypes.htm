Types d'utilisateur
Oracle, en tant que système de gestion de bases de données objet-relationnel, permet de définir des types de données supplémentaires et de les utiliser avec le modèle relationnel.
Utilisez Type d'objet pour créer un type complexe représentant une abstraction des entités réelles traitées par votre application.
Utilisez Type de tableau pour créer un ensemble trié d'éléments de données du même type.
Utilisez Type de table pour créer un ensemble non trié d'éléments de données du même type.
Utilisez Schéma XML pour créer des types et des éléments de stockage complexes destinés aux documents XML basés sur le schéma XML. Ce type d'objet est disponible pour Oracle versions 9.2 et supérieures.
CONSEIL Oracle recommande d'utiliser la console Enterprise Manager 10g pour l'administration et les diagnostics.
