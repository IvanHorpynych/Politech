Oracle LogMiner(tm) Viewer
 
Utilizzare LogMiner Viewer per eseguire query su redo log file e facilitare quindi l'analisi dell'attività di modifica eseguita in precedenza sul database.

Con LogMiner Viewer è possibile effettuare con facilità le operazioni riportate di seguito.
Specificare i criteri delle query per scegliere i dati da recuperare dai redo log file.
Visualizzare i dati recuperati dai redo log file, comprese le istruzioni SQL redo e undo.
Specificare altri redo log file su cui eseguire query.
Specificare le colonne da visualizzare e il loro ordine.
Salvare i criteri e i risultati delle query.

Per iniziare, fare clic su Crea query, quindi immettere i criteri della query e fare clic sul pulsante Trova adesso.
