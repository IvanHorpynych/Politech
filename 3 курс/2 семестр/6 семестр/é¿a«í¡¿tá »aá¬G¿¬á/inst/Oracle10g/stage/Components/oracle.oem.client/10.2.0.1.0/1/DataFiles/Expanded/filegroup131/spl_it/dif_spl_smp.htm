DB Diff consente di confrontare le definizioni degli oggetti. È possibile confrontare le definizioni di due database, di due baseline, di un database e di una baseline oppure di due schemi dello stesso database o della stessa baseline.

Successivamente è possibile utilizzare la Sincronizzazione guidata di DB Diff per modificare le definizioni degli oggetti del database o dello schema in modo che le definizioni degli oggetti di tali elementi corrispondano.
