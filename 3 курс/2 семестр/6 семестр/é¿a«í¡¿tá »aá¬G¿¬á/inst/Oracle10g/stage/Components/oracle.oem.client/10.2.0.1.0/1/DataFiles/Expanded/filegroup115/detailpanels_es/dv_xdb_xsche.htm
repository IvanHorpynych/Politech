Objetos Basados en Esquema XML
Gestión de Bases de Datos XML>Objetos Basados en Esquema XML
Esquema XML es un lenguaje de definición de esquemas (también en XML) que se puede utilizar para describir la estructura y la semántica de los documentos de instancia XML que se ajusten.
Oracle 9.2 permite:
Registrar un esquema XML con la base de datos bien mediante un recurso de base de datos XML existente o mediante un archivo de esquema de base de datos XML al que se puede acceder de forma local o a través de Internet.
Crear un almacenamiento estructurado basado en el esquema XML.
Puede realizar esta operación de dos formas:
Crear una tabla basada en la estructura definida en el esquema XML
Crear una tabla con columna que esté basada en la estructura definida en el esquema XML
Las ventajas de crear almacenamiento basado en el esquema XML son:
  • Los documentos de instancia XML guardados en este almacenamiento ocupan significativamente menos espacio.
  • El acceso a datos concretos, expresados a través de XPath, está muy optimizado.
  • Las operaciones de actualización de partes del documento de instancia XML se han mejorado enormemente.
Ajustar datos existentes en formatos XML mediante la creación de vistas basadas en el esquema XML
También puede crear índices funcionales en tablas basadas en el esquema XML mediante expresiones XPath.
