Objetos de Metadatos de OLAP
Gestión de OLAP > Objetos de Metadatos de OLAP
Para realizar un procesamiento analítico, el almacén de datos debe estar preconfigurado con objetos de metadatos de OLAP y el acceso al mismo debe estar optimizado mediante la creación de vistas materializadas. Los objetos de metadatos de OLAP son los siguientes:

  • Objetos de Dimensión: Organizan tablas de dimensiones de almacenes de datos en niveles, atributos y jerarquías. Para crear estos objetos, inicie el Asistente de Dimensiones.

  • Cubos: Son representaciones lógicas de datos multidimensionales y deben incluir al menos una dimensión. Para crear cubos, inicie el Asistente de Creación de Cubos.

  • Carpetas de Medidas: Organizan medidas en grupos que se pueden gestionar. Para crear carpetas de medidas, inicie el recuadro de diálogo Carpeta de Medidas.
Cuando un cubo esté seleccionado en el árbol del navegador, haga clic con el botón derecho del mouse en él para iniciar las siguientes herramientas:
  • Asistente de Asesor de Resúmenes: Este asistente recomienda vistas materializadas que optimizan el rendimiento de las aplicaciones cliente de OLAP que acceden a los datos definidos por un cubo.

  • Cube Viewer: Esta aplicación muestra los datos multidimensionales que representa el cubo.
