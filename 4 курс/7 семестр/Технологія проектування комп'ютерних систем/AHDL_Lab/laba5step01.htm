Шаг 1

1) Запустите AVHDL.

2) Загрузите проект laby (Для этого необходимо в меню, которое появится после запуска AVHDL выбрать необходимое имя проекта)

3) Создайте файл, где будет рисоваться граф переходов автомата, используя мастер создания.

а) Сделайте двойной щелчок на метке Add New File в окне Design Browser.
б) В появившемся окне выберите пунк Create State Diagram File Using Wizard (создать файл состояний автомата с использованием помощника)
 
 
Для продолжения нажмите Ok
