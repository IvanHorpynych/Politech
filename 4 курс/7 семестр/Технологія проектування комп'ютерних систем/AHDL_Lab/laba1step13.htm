Шаг 13

Теперь необходимо задать формы входных сигналов. Для этого нужно установить курсор на входной порт (к примеру in1), нажать правую кнопку мыши и в контекстном меню выбрать пункт Stimulators...

После этого на экране появится следующее окно

Выберете в меню Simulator type опцию Clock, установите в появившемся окне нужную частоту (или период), фазу входного сигнала.

Нажмите Apply когда все будет готово. Аналогичные действия проведите со вторым портом ввода (установите для него частоту в два раза меньше).
