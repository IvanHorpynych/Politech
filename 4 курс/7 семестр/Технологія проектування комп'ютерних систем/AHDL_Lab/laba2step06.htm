Шаг 6

Если компиляция прошла без ошибок, то теперь необходимо протестировать созданую схему. Для этого последовательно выполните следующие действия:

  1. Установите модулю Task1 Top статус Level;
  2. запустите инициализацию моделирования;
  3. создайте окно Wave form;
  4. добавтьте в него необходимые для тестирования сигналы (используйте пункт Add Signal контекстного меню);
  5. установите входным сигналам необходимые частоты;
  6. запустите моделирование (рекомендуется использовать пошаговое: Trace For (по умолчанию 100нс)).
Проанализируйте полученные временные диаграммы с учетом задержек элементов, то есть необходимо учитывать, что правильный сигнал установиться на выходе после определенного промежутка времени равному максимольному времени прохождения сигнала по цепи. В нашем случае он составляет 40нс:
