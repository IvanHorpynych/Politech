Шаг 10

Теперь нужно проверить работоспособность элементов по временным диаграммам.

Сначала надо выбрать тестируемый элемент (так как файл может содержать несколько описаний различных элементов) установив его на самый высокий уровень. Для этого необходимо выбрать описание требуемого элемента, нажать правую кнопку мыши и выбрать пункт Set as Top-Level.

После этого в окне Design Browser в секции Top-Level selection появится название требуемого элемента.
