Шаг 1

1) Запустите AVHDL.

2) Загрузите проект laby (Для этого необходимо в меню, которое появится после запуска AVHDL выбрать необходимое имя проекта).

3) Добавьте в проект новый пустой файл ресурса используя мастер создания файла:

а) Сделайте двойной щелчок на метке Add New File в окне Design Browser.
б) В появившемся окне выберите пункт Create EmptyVHDL Source File (создать пустой файл).

Для продолжения нажмите Ok
