Следующий этап - компилирование файла описания.

Для этого необходимо запустить компилятор. Это можно сделать нажатием кнопки в панели инструментов или выбрать Compile в меню Design. Если все было сделано правильно, то в окне Console будет показан следующий отчет о результатах компиляции:

Compile Entity "and2"
Compile Architecture "and2" of Entity "and2"
Compile success 0 Errors 0 Warnings Analysis time : 2.0 [s]
