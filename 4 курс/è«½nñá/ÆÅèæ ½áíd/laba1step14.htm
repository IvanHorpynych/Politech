Шаг 14

Теперь необходимо построить совмещенные диаграммы входных и выходных сигналов. Для этого нужно нажать кнопку Run For.

Это позволит просчитать схему на 100нс в перед. Нажмите кнопку Run For несколько раз так, чтобы протестировать элемент на всей последовательности входных сигналов.

Из диаграммы видно, что моделируемый элемент AND имеет задержку. Для ее определения необходимо расставить метки на диаграмме.

Нужно подвести курсор к месту установки метки и, нажав правую кнопку мыши, в контекстном меню выбрать Add Cursor.

После расстановки меток на диаграмме можно проверить длительность задержек, которые необходимо было задать элементу.
