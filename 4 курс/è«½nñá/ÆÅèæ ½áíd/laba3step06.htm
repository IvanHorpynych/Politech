Шаг 6

Начнем тестирование:

1) Нажмети два раза на кнопку Run For для того, чтобы промоделировать состояние j=0, k=0,

Как видно из диаграмм при j=0,k=0, состояние не меняется.

2) Нажмите на клавишу j и на клавишу Run For (тестируется состояние j=1, k=0).

Как видно из диаграмм при j=1,k=0, тригер устанавливается в состояние 1,0.

3) Нажмите клавишу k, а потом Run For.(тестируется состояние j=1, k=1).

При j=1,k=1 тригер меняет свое состояние на противополжное.

4) Установите состояние j=0, k=1 (для этого нажмите на клавишу j); промоделируйте на некотором отрезке времени:

При j=0,k=1 тригер устанавливается в состояние 0,1.

Работоспособность сброса проверяется точно таким же способом.

Как видно из диаграмм, построенный нами тригер работает правильно.

Если построенные вами диаграммы совпадают с теми, которые должны иметь место при использовании заданного тригера, то данный шаг можно счиать законченным.
